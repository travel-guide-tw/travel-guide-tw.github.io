# 盧加諾湖 (義大利語：Lago di Lugano)

![Lago di Lugano photo by Huang An Sheng](https://i.imgur.com/UoUCTpN.jpg)

盧加諾湖 (義大利語：Lago di Lugano) 位於瑞士東南部，處瑞士和義大利兩國交界處的一個湖泊。湖名來源於瑞士城市盧加諾。位於馬焦雷湖和科莫湖之間。盧加諾湖是一個知名的觀光地。
