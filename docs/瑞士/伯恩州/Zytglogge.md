# Zytglogge

![Zytglogge photo by Huang An Sheng](https://i.imgur.com/jHenIgJ.jpg)

位於伯恩市中心的鐘樓，是個相當顯著的地標，也是許多去伯恩旅行的旅行團會選擇的集結點。

- 地址 Bim Zytglogge 1, 3011 Bern, Switzerland
