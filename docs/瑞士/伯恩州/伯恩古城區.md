# 伯恩古城區

![Bern photo by Huang An Sheng](https://i.imgur.com/BFZXQ3A.jpg)

伯恩是瑞士聯邦政府所在地兼事實上的首都。瑞士伯恩的舊城區，建造於三面環河的小山上。目前仍保留大部分中世紀的面貌，並與現代城市的機能巧妙地融合。1983 年被列為世界文化遺產。當前老城內的眾多建築以及整個老城都被指定為瑞士國家級文化財產。
