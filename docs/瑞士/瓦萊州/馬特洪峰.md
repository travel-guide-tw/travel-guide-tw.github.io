# 馬特洪峰 (Matterhorn)

![Matterhorn photo by Huang An Sheng](https://i.imgur.com/7udczKL.jpg)

馬特洪峰，也稱切爾維諾峰，是阿爾卑斯山脈中最著名的山峰。馬特洪峰的位置在瑞士、義大利邊境，馬特洪峰的名稱是由德語「Matt」（意為草地）和「Horn」（意為角，或呈錐狀像一隻角一樣的山峰）組成。
