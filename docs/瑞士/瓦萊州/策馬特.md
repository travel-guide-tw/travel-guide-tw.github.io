# 策馬特 (Zermatt)

![Zermatt photo by Huang An Sheng](https://i.imgur.com/WmZbTE3.jpg)

策馬特 (Zermatt) 位於馬特洪峰 Matterhorn 脚下，其旅遊業發展與世界上最著名山峰之一的馬特洪峰息息相關。這個渡假勝地禁止車輛通行，致使它很好地保留清新的原貌，爲旅客的觀光提供了無限的可能性。
