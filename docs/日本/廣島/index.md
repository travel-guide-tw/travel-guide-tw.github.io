# 廣島

## 歷史與和平

廣島縣位於日本中國地區，面臨瀨戶內海，擁有豐富的歷史文化和美麗的自然風光。廣島因原子彈爆炸而聞名於世，但這座城市在戰後迅速重建，成為一個現代化的都市，同時也致力於推廣世界和平。

## 必訪景點

- **[原爆圓頂館](./景點/原爆圓頂館)：** 歷史的見證與和平的象徵。
- **[和平紀念公園](./景點/和平紀念公園)：** 包含和平紀念資料館、和平紀念碑等。
- **[宮島 (嚴島神社)](./景點/宮島)：** 瀨戶內海上的海上鳥居。
- **[廣島城](./景點/廣島城)：** 又名「鯉城」，江戶風格的雄偉城堡。
- **[縮景園](./景點/縮景園)：** 四百年歷史的美麗大名庭園。
- **[千光寺](./景點/千光寺)：** 位於尾道，俯瞰瀨戶內海絕景。

## 交通與美食

- **[交通資訊](./交通/)：** 包含路面電車及觀光循環巴士資訊。
- **[在地美食](./廣島燒.md)：** 必嚐的[廣島燒](./廣島燒.md)。

## 鄰近區域推薦

廣島作為中國地區的中心，方便前往周邊特色城市：

- **[岡山縣](../岡山/)：** 搭乘新幹線僅需 35 分鐘，體驗晴天之國。
- **[愛媛縣](../愛媛/)：** 從尾道出發，經由「島波海道」單車跨海前往四國。
- **[山口縣](../山口/)：** 探索海上鳥居與錦帶橋。是著名的賞櫻與歷史名勝（內容編寫中）。

## 溫馨提示

- 參觀原爆圓頂館和和平紀念公園時，請保持肅靜，尊重逝者。
- 前往宮島時，請務必查詢潮汐時間，以便欣賞到最美麗的海上鳥居景色。
- 廣島燒是廣島地區最具代表性的美食之一，與大阪燒不同，廣島燒採用分層疊放的製作方式，並加入蕎麥麵，口感豐富。可以前往御好燒村，一次品嚐多家名店的口味。
