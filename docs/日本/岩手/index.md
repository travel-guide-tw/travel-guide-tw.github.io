# 岩手

## 廣闊的大地與世界遺產

岩手縣是日本面積第二大的行政區（僅次於北海道），擁有壯麗的海岸線、雄偉的山脈以及被列為世界文化遺產的平泉文化。

## 必訪景點

- **中尊寺：** 位於平泉町，其中的「金色堂」貼滿了金箔，是奧州藤原氏繁榮時期的象徵。
- **龍泉洞：** 日本三大鐘乳石洞之一，以其深邃且清澈的藍色地底湖（龍藍）而聞名。
- **嚴美溪：** 以奇岩怪石和急流著稱，這裡的「郭公糰子」（飛天糰子）非常有趣。

## 交通資訊

從東京搭乘東北新幹線至盛岡站約需 2 小時 10 分鐘。在縣內移動，可以利用 JR 鐵路、三陸鐵路或租車。

## 溫馨提示

- 盛岡市以「盛岡三大麵」（冷麵、炸醬麵、碗子蕎麥麵）聞名，一定要挑戰看看。
- 岩手縣面積廣大，景點之間距離較遠，建議租車自駕。
