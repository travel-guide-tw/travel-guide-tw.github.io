# 攀登富士山 (靜岡側路線)

富士山橫跨山梨縣與靜岡縣，其中四大主要登山路線中，有三條位於靜岡縣側：**須走線**、**御殿場線**與**富士宮線**。

## 靜岡側路線特色

與山梨縣側的吉田線相比，靜岡側的路線各有特色：

- **富士宮線**：起點海拔最高，是登頂路徑最短的路線。
- **須走線**：擁有豐富的自然景觀與著名的「砂走」下山體驗。
- **御殿場線**：海拔落差最大，最具挑戰性，適合資深登山者。

## 靜岡側交通資訊

前往靜岡側各合目的交通主要依靠夏季限定的登山巴士 (通常為 7/10 - 9/10 營運)：

### 1. 富士宮線 (Fujinomiya Trail)

- **主要起點**：JR 新富士站 (新幹線)、JR 富士站、JR 富士宮站。
- **巴士營運**：富士急靜岡巴士。
- **票券建議**：可購買「富士登山巴士自由乘車券」(3 日有效)。
- **自駕限制**：登山期間禁止私家車進入，須在「水塚公園」轉乘接駁巴士。

### 2. 須走線 (Subashiri Trail)

- **主要起點**：JR 御殿場站、小田急新松田站。
- **巴士營運**：富士急摩比斯巴士。

### 3. 御殿場線 (Gotemba Trail)

- **主要起點**：JR 御殿場站。
- **巴士營運**：富士急摩比斯巴士。

**詳細交通票券與路線請參考**：👉 **[靜岡交通指南](../交通/index)**
