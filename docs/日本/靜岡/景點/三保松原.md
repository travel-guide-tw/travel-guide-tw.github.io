# 三保松原 (Miho no Matsubara)

三保松原位於靜岡市清水區，是一處長約 7 公里的海岸線，沿岸種植了約 3 萬棵松樹。這裡是「富士山世界文化遺產」的構成資產之一，以「白砂青松」與富士山交織的絕景聞名。

## 主要特色

- **富士山絕景**：這裡與歌山縣的「天之橋立」、宮城縣的「松島」齊名，雖然不屬於傳統日本三景，但其富士山景觀極具代表性。
- **羽衣傳說**：傳說曾有仙女在此降臨，將羽衣掛在松樹上。境內有一棵被稱為「羽衣之松」的古松。
- **神之步道**：連接御穗神社與三保松原的木製步道，兩旁古松參天，氛圍莊嚴。

## 旅遊建議

- **御穗神社**：參觀完松原後，可以順著「神之步道」前往御穗神社參拜。
- **富士山世界遺產中心**：附近有「靜岡市三保松原文化創造中心 (Miho Navi)」，可以深入了解相關歷史與藝術作品。

## 交通資訊

- **巴士**：從 JR「清水站」搭乘前往「三保水族館」方向的巴士，在「三保松原入口」下車後步行約 10-15 分鐘。
- **水上巴士**：可以從清水港搭乘水上巴士前往三保，從海上欣賞富士山。
