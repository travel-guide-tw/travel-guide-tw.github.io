# 靜岡

## 富士山的南大門

靜岡縣位於日本本州中部，坐擁富士山南麓，擁有豐富的自然景觀、著名的溫泉以及全日本第一的茶葉產量。這裡不僅可以從不同角度欣賞富士山，還有美麗的伊豆半島、駿河灣海景以及歷史悠久的古道。

## 必訪景點

- **[富士山](./活動/攀登富士山)：** 從靜岡側體驗不同的登山路線與壯闊海景。
- **[三保松原](./景點/三保松原)：** 著名的「富士山世界文化遺產」構成資產之一。
- **[熱海溫泉](./景點/熱海溫泉)：** 著名的溫泉勝地與花火大會。
- **[富士山本宮淺間大社](./景點/富士山本宮淺間大社)：** 富士山信仰的中心。

## 交通資訊

- **[交通概覽](./交通/index)：** 靜岡縣交通便利，可搭乘新幹線前往靜岡、濱松、熱海等地，也有前往富士山各登山口的巴士。

## 溫馨提示

- 靜岡縣的茶葉非常出名，推薦前往茶園體驗採茶或品嚐正宗抹茶。
- 攀登富士山靜岡側路線（須走、御殿場、富士宮）時，請務必確認開放時間與預約規定。

## 相關地區
