# 大內宿

大內宿位於福島縣南會津郡下鄉町，是江戶時代連接會津若松與日光的重要驛站（宿場町）。這裡至今仍保留著 30 多座茅草屋頂的傳統建築，被指定為日本國家重要傳統建築物群保存地區。

![大內宿](https://upload.wikimedia.org/wikipedia/commons/e/e3/Ouchijuku_2006-11-12.jpg)
_圖片來源：Wikimedia Commons_

## 景點特色

走在寬闊的街道上，兩旁是古色古香的茅草屋，讓人彷彿回到了江戶時代。這裡的建築現在多作為土產店、餐廳或民宿經營。

## 門票資訊

- **聚落參觀：** 免費。
- **大內宿町展覽館：** 成人 250 日圓，中小學生 150 日圓。

## 必吃美食

- **高遠蕎麥麵（大蔥蕎麥麵）：** 這裡最著名的美食。吃法非常獨特，不使用筷子，而是用一整根大蔥撈起麵條來吃，並同時咬一口大蔥作為辛香料。

## 交通手段

### 大眾運輸

1. **搭乘新幹線：** 至「郡山站」。
2. **轉乘 JR 磐越西線：** 至「會津若松站」。
3. **轉乘會津鐵道：** 至「湯野上溫泉站」。
4. **搭乘巴士：** 從湯野上溫泉站搭乘觀光巴士「猿遊號」（猿游号）前往大內宿（車程約 20 分鐘）。
   - **建議：** 可購買「大內宿共通套票」（約 2,200 日圓），包含鐵道與巴士費用。

## 溫馨提示

- 冬季的大內宿覆蓋著厚厚的白雪，景色非常夢幻，每年 2 月還會舉辦雪祭。
- 建議在早上或傍晚遊客較少時前往，更能感受古鎮的寧靜氛圍。
