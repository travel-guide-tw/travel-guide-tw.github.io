# 東大寺

東大寺是奈良最具代表性的寺院，其「大佛殿」是世界上最大的木造建築之一，殿內供奉著高達 15 公尺的巨大青銅佛像「奈良大佛」。這座寺院不僅是世界文化遺產，也是日本文化與藝術的瑰寶。

- 地址：〒 630-8211 奈良市雜司町 406-1
