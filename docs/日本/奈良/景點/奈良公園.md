# 奈良公園

奈良公園是一座歷史悠久的都市公園，最著名的莫過於散養在園內的上千頭梅花鹿，牠們被視為神的使者。遊客可以購買鹿仙貝與鹿群互動。公園佔地廣闊，連結了東大寺、春日大社與興福寺等重要古蹟。

- 地址：〒 630-8211 奈良市雜司町、春日野町一帶
