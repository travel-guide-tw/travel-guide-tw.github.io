# 奈良

## 古都巡禮

奈良，古名大和，是日本歷史的搖籃，也是日本文化和佛教的發源地之一。這座城市擁有豐富的歷史遺產，包括眾多的寺廟、神社和古蹟，其中許多已被聯合國教科文組織列為世界遺產。

## 必訪景點

- **[東大寺](./景點/東大寺)：** 世界上最大的木造建築。
- **[春日大社](./景點/春日大社)：** 日本最重要的神社之一。
- **[奈良公園](./景點/奈良公園)：** 與可愛的梅花鹿互動。

## 交通資訊

- **[交通概覽](./交通/)：** 前往奈良的交通十分便利，您可以搭乘 JR 或近鐵電車抵達。在市內觀光，奈良交通巴士是主要的交通工具。

## 鄰近區域推薦

奈良與京都、大阪緊鄰，是關西三都旅遊的重要一環：
- **[京都府](../京都/)：** 延續古都歷史之旅。
- **[大阪府](../大阪/)：** 感受現代都市活力。
- **[和歌山縣](../和歌山/)：** 探訪高野山等佛教聖地。

## 貼心提醒

- 奈良公園的梅花鹿雖然溫馴，但請勿餵食牠們人類的食物，以免影響牠們的健康。
- 參觀寺廟和神社時，請保持肅靜，並遵守相關規定。
- 建議購買奈良・斑鳩一日券，可以無限次搭乘指定區域內的交通工具，並享有部分景點的門票優惠。
