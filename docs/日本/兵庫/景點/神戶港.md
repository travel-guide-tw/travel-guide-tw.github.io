# 神戶港

神戶港是日本最重要的國際貿易港口之一，以其美麗的夜景聞名。「神戶港塔」與「美利堅公園」是其地標，周邊的摩塞克廣場 (MOSAIC) 擁有眾多的商店與餐廳。坐在港邊吹海風、欣賞神戶大橋的燈光，是神戶最浪漫的體驗。

- 地址：〒 650-0042 神戶市中央區波止場町
