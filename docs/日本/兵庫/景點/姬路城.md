# 姬路城

姬路城是日本首批列入世界文化遺產的古蹟，因其潔白的外牆與優美的造型宛如展翅欲飛的白鷺，故又享有「白鷺城」的美譽。它是日本現存少數保有完整木造天守閣的城堡，也是日本國寶之一，深受國內外遊客喜愛。

- 地址：〒 670-0012 姬路市本町 68
