# 有馬溫泉

有馬溫泉是日本三大古湯之一，其歷史可追溯至神話時代。這裡以含有豐富鐵質與鹽分的「金泉」以及透明清澈的「銀泉」聞名。溫泉街上充滿古典氛圍，有許多傳統旅館、公共浴場與特色小店，是神戶郊區極受歡迎的休閒勝地。

- 地址：〒 651-1401 神戶市北區有馬町
