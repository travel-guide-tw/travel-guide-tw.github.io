# 天上山公園

搭乘富士山景觀纜車前往天上山公園，可以俯瞰富士山和河口湖的美景。

- 地址 〒401-0303 山梨県南都留郡富士河口湖町浅川 1163-1
- 電話號碼 +81555723168
- 營業時間 9:00 ～ 17:00
