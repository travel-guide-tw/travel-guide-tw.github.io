# 天上山公園 (Tenjozan Park) / 富士山全景纜車

天上山公園位於河口湖東岸的天上山，可以透過搭乘「富士山全景纜車」輕鬆抵達。這裡以太宰治的小說《卡齊卡齊山》(Kachi Kachi Yama) 為背景，公園內隨處可見可愛的兔子與狸貓裝飾。

## 景點特色

- **富士山全景展望台**：從海拔 1,075 公尺的展望台，可以俯瞰整個河口湖市街與雄偉的富士山。
- **天上之鐘**：與富士山相對的神社與鐘，是熱門的拍照打卡點。
- **狸貓茶屋**：提供印有富士山圖案的烤糰子 (狸貓糰子) 與各種紀念品。
- **兔子神社**：祈求腿部健康與觀光安全的可愛神社。
- **扔土器 (かわらけ投げ)**：將小陶片扔過圓圈，祈求心願達成。

## 基本資訊

- **地址**：〒401-0303 山梨県南都留郡富士河口湖町浅川 1163-1
- **電話**：+81 555-72-0363 (纜車營運商)
- **營業時間**：09:30 - 16:00 (平日) / 09:30 - 17:00 (週末及假日)。
- **纜車票價**：成人來回約 900 日圓 / 兒童約 450 日圓。

## 交通方式

- 搭乘**河口湖周遊巴士 (紅線)** 至「遊覽船・山纜車站」下車，步行約 2 分鐘即可到達纜車站。
