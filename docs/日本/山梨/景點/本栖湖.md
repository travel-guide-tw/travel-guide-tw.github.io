# 本栖湖 (Lake Motosu)

![本栖湖與富士山](https://upload.wikimedia.org/wikipedia/commons/c/c9/Mount_Fuji_from_Lake_Motosu.jpg)

<!-- 圖片來源：Wikimedia Commons -->

本栖湖是富士五湖中深度最深、透明度最高的湖泊。日圓千圓紙鈔背面的富士山倒影圖案，就是在本栖湖北岸所拍攝的。

## 景點特色

- **千圓紙鈔取景地**：位於北岸的中之倉峠是拍攝千圓紙鈔風景的起點。
- **水上活動**：水質清澈，非常適合潛水、風帆及皮划艇。
- **富士芝櫻祭**：本栖湖周邊在春天會舉辦著名的富士芝櫻祭。

## 基本資訊

- **地址**：山梨県南都留郡富士河口湖町・身延町
- **交通方式**：搭乘**鳴澤・精進湖・本栖湖周遊巴士 (藍線)** 至終點站「本栖湖」下車。
