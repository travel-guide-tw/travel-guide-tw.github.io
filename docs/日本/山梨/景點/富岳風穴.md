# 富岳風穴 (Fugaku Wind Cave)

![富岳風穴入口](https://upload.wikimedia.org/wikipedia/commons/9/92/Fugaku_Fuketu_Entrance_20161025.jpg)

<!-- 圖片來源：Wikimedia Commons -->

富岳風穴是青木原樹海中最大的溶岩洞穴。洞穴內部全年氣溫維持在攝氏 3 度左右，即使在夏天也感覺非常涼爽。這裡曾被用作天然冰箱來儲存蠶繭。

## 景點特色

- **天然冰箱**：洞內可以看到過去用來儲存蠶繭和種子的遺跡。
- **溶岩景觀**：可以近距離觀察溶岩流動形成的特殊地質景觀。
- **樹海步道**：風穴周邊有步道連接至鳴澤冰穴，可以漫步於青木原樹海中。

## 基本資訊

- **地址**：〒401-0332 山梨県南都留郡富士河口湖町西湖青木ヶ原 2011
- **電話**：+81 555-85-2300
- **營業時間**：09:00 - 17:00 (隨季節調整)。
- **門票**：成人約 350 日圓 (與鳴澤冰穴有套票)。

## 交通方式

- 搭乘**西湖周遊巴士 (綠線)** 或 **藍線巴士** 至「富岳風穴」站下車即達。
