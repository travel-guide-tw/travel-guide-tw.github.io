# 河口湖香草館 (Kawaguchiko Herb Hall)

河口湖香草館位於河口湖南岸，是一個以香草為主題的觀光景點。館內販售各種香草產品、精油、香氛，後方還有精緻的香草花園。

## 景點特色

- **香草花園**：後方的花園種植了兩百多種香草，四季皆可參觀。
- **手作體驗**：提供押花、乾燥花製作等手作課程。
- **香草霜淇淋**：館內販售的薰衣草霜淇淋是這裡的人氣商品。

## 基本資訊

- **地址**：〒401-0301 山梨県南都留郡富士河口湖町船津 6713-18
- **電話**：+81 555-72-3087
- **營業時間**：09:00 - 18:00 (冬季時間可能調整)。
- **門票**：免費參觀。

## 交通方式

- 搭乘**河口湖周遊巴士 (紅線/綠線)** 至「河口湖香草館」站下車即達。
