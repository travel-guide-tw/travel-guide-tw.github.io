# 河口湖紅葉迴廊

河口湖地區知名的賞楓景點。大約在每年的 11 月上旬或中旬，楓葉會逐漸轉紅。多棵楓樹在旁排列，有著相當壯觀的楓葉景色。

- 地址 〒401-0304 山梨県南都留郡富士河口湖町河口
- 電話號碼 +81555723168
