# 山梨寶石博物館 (Yamanashi Gem Museum)

![山梨寶石博物館外觀](https://upload.wikimedia.org/wikipedia/commons/3/31/Yamanashi_Gem_Museum.JPG)

<!-- 圖片來源：Wikimedia Commons -->

山梨縣是日本著名的寶石與珠寶加工中心，而山梨寶石博物館則展示了來自世界各地的三千多件珍貴原石、寶石及珠寶作品。

## 景點特色

- **巨大原石**：館內展示了令人驚嘆的巨型水晶與原石。
- **世界級珠寶**：展出各種切割完美的鑽石、紅寶石等珍貴珠寶。
- **博物館商店**：販售各種天然石飾品與紀念品。
