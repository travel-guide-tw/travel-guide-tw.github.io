# 久保田一竹美術館 (Itchiku Kubota Art Museum)

![久保田一竹美術館建築](https://upload.wikimedia.org/wikipedia/commons/a/a5/Itchiku_Kubota_Art_Museum_2018a.jpg)

<!-- 圖片來源：Wikimedia Commons -->

久保田一竹美術館致力於展示染色藝術家久保田一竹的作品，他復興了失傳已久的「辻花 (Tsujigahana)」絲綢染色技術。美術館本身的建築與庭園也極具藝術價值。

## 景點特色

- **和服藝術**：館內展示了以富士山、四季為主題的宏偉和服作品，精美絕倫。
- **獨特建築**：主館由樹齡超過千年的柏樹建成，結構奇特，融合了自然與人文。
- **禪意庭園**：園內設有瀑布、小徑和具有異國情調的裝飾，非常適合散心。
