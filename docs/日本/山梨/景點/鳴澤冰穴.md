# 鳴澤冰穴 (Narusawa Ice Cave)

![鳴澤冰穴入口](https://upload.wikimedia.org/wikipedia/commons/5/58/Narusawa_Hyoketsu_Entrance_20161025.jpg)

<!-- 圖片來源：Wikimedia Commons -->

鳴澤冰穴是一個長度約 153 公尺的環狀溶岩洞穴。洞內終年結冰，即使在盛夏也能看到巨大的冰柱。洞穴高度較低，有些地方需要彎腰甚至爬行，充滿探險感。

## 景點特色

- **萬年冰柱**：由上方滴落的水滴結成，場面壯觀。
- **溶岩隧道**：展示了富士山噴發後溶岩冷卻形成的特殊構造。
- **探險體驗**：洞內道路狹窄崎嶇，是一場短暫的地下冒險。

## 基本資訊

- **地址**：〒401-0320 山梨県南都留郡鳴沢村 8533
- **電話**：+81 555-85-2301
- **營業時間**：09:00 - 17:00 (隨季節調整)。
- **門票**：成人約 350 日圓 (與富岳風穴有套票)。

## 交通方式

- 搭乘**藍線巴士** 至「鳴澤冰穴」站下車即達。 (綠線巴士亦可抵達附近的富岳風穴再步行前往)
