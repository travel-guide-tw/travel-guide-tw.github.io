# 河口湖美術館 (Kawaguchiko Museum of Art)

河口湖美術館坐落於河口湖北岸，以展示與富士山相關的繪畫、攝影作品而聞名。美術館周圍有著寬闊的草坪，是欣賞湖景與山景的好地方。

## 景點特色

- **富士山攝影展**：館內經常舉辦以富士山為主題的攝影比賽得獎作品展。
- **草坪廣場**：美術館前的草坪可以自由休憩，是觀賞河口湖的絕佳視角。
- **咖啡廳**：館內設有景觀咖啡廳，可以一邊喝咖啡一邊欣賞湖光山色。
