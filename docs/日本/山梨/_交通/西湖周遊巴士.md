# 西湖周遊巴士 (綠線 / Green Line)

綠線（Green Line）前往自然資源豐富的西湖地區，適合喜歡寧靜景致、傳統建築與自然洞穴探險的旅客。

- **起點**：河口湖車站
- **班次**：約每 30-60 分鐘一班
- **主要特色**：風景比紅線更原始，有著名的合掌造村落與溶岩洞穴。
