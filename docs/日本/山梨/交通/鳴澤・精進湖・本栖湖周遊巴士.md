# 鳴澤・精進湖・本栖湖周遊巴士 (藍線 / Blue Line)

藍線（Blue Line）帶您前往富士五湖中最深處的精進湖與本栖湖。這條路線最能感受到大自然的壯闊與安寧。

- **起點**：河口湖車站
- **班次**：班次較少（約每 2 小時一班），建議先查好時刻表。
- **主要特色**：行經本栖湖（千圓紙鈔景色）與精進湖（抱子富士）。
