# 妻籠宿

_圖片來源：Wikimedia Commons_

妻籠宿位於長野縣木曾郡南木曾町，是中山道 69 次中的第 42 號宿場。
...

## 交通資訊

從 JR 南木曾站搭乘巴士約 10 分鐘，或步行約 40 分鐘即可抵達。詳細班次請參考 **[南木曾町營巴士官方網站](https://www.town.nagiso.nagano.jp/kankou/bus_taxi/index.html)**。可搭配 **[JR Pass](/日本/愛知/交通/JR Pass)** 規劃行程。
