# 松島

松島位於宮城縣松島灣，由 260 多座奇松怪石的小島組成，與京都的天橋立、廣島的嚴島並稱為「日本三景」。

[松島官方圖片庫 (松島觀光協會)](https://www.matsushima-kanko.com/miryoku/photo-gallery/)
_圖片來源：(一社) 松島觀光協會_

## 景點特色

松島的美景自古以來就受到文人墨客的讚美。遊客可以搭乘遊覽船穿梭在各個小島之間，或是前往五大堂、瑞嚴寺等歷史建築參觀。

## 門票資訊

- **松島灣觀光遊船：** 成人約 1,500 日圓（航程約 50 分鐘）。
- **瑞嚴寺：** 成人 700 日圓，兒童 400 日圓。
- **五大堂：** 免費參觀。
- **福浦島：** 通行費 200 日圓（經由朱紅色福浦橋）。

## 交通手段

### 火車 (最推薦)

- **從仙台站出發：** 搭乘 JR 仙石線至「松島海岸站」，車程約 40 分鐘。
  - **注意：** 請在「松島海岸站」下車，而非「松島站」，前者距離主要景點較近。

## 溫馨提示

- 建議在傍晚時分欣賞松島的夕陽，景色非常迷人。
- 當地的烤牡蠣和毛豆泥（Zunda）甜點是必嚐美食。
