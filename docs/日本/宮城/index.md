# 宮城

## 杜之都與松島之美

宮城縣位於東北地區的中心，其首府仙台市被譽為「杜之都」（森林之都）。宮城縣擁有日本三景之一的松島，以及豐富的歷史文化和美味的牛舌。

## 必訪景點

- **[松島](./景點/松島)：** 由 260 多座島嶼組成的海灣，是日本三景之一，可以搭乘遊覽船欣賞壯麗的海景。
- **仙台城跡（青葉城）：** 戰國大名伊達政宗所建的城堡遺址，可以俯瞰仙台市區。
- **藏王狐狸村：** 位於白石市山區，可以近距離觀察和餵食可愛的狐狸。

## 交通資訊

從東京搭乘東北新幹線至仙台站只需約 1.5 小時。仙台市內交通便利，可以利用地鐵、巴士或觀光巴士「Loople 仙台」。

## 溫馨提示

- 來到仙台一定要品嚐當地的名產——炭烤牛舌。
- 松島的牡蠣非常有名，冬季是品嚐牡蠣的最佳季節。
