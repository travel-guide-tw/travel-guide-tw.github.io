# 大湯沼川天然足湯

大湯沼川是從大湯沼溢流而出的溫泉河，流經原始森林。遊客可以在森林中坐下來，將雙腳浸泡在溫暖的河水中，享受天然的「森林浴」與「足湯」。

## 景點特色

- **純天然溫泉河：** 河水帶著濃郁的白濁色（硫磺泉），溫度適中，非常舒適。
- **森林秘境：** 步道穿梭在茂密的森林中，秋季紅葉環繞，是登別溫泉最放鬆的角落。
- **身心療癒：** 步行完地獄谷與大湯沼後，在此泡腳可有效緩解疲勞。

## 參觀資訊

- **門票：** 免費使用。
- **開放時間：** 全天（建議白天前往，森林中無路燈）。

## 交通手段

- **步行：**
  - 從大湯沼步行約 15 分鐘。
  - 從溫泉街入口（道南巴士總站附近）步道口步行約 20 分鐘。詳見 **[北海道巴士指南](../../交通/巴士)**。

## 溫馨提示

- **自備毛巾**：現場不提供毛巾，建議隨身攜帶小毛巾擦腳。
- **注意防滑**：溪邊石頭可能濕滑，移動時請小心。
- **避開雨後**：若剛下過大雨，河水流量增加且溫度可能下降。
