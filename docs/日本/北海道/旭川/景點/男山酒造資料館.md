# 男山酒造資料館 (Otokoyama Sake Brewery & Museum)

男山是旭川最著名的清酒品牌，擁有超過 350 年的歷史。男山酒造資料館展示了江戶時代至今的造酒工具、歷史文獻與繪畫，是了解日本清酒文化的重要地點。

## 景點特色

- **歷史展覽：** 館內收藏了許多江戶時代的釀酒器具，以及描繪當時飲酒文化的浮世繪。
- **試飲與販售：** 遊客可以免費或付費品嚐多款男山的經典清酒，包括屢獲國際金獎的大吟釀。
- **延命水：** 釀造男山清酒所使用的水源是大雪山連峰的伏流水，遊客可以現場飲用這口傳說中的「延命水」。
