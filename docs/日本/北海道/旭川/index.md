# 旭川

## 北海道的心臟

旭川是北海道第二大城市，位於北海道的中心位置。這裡以其優質的旭川拉麵、精緻的工藝品以及著名的旭山動物園而聞名。

## 必訪景點

- **[旭山動物園](./景點/旭山動物園)：** 日本最受歡迎的動物園之一，以其「行動展示」方式讓遊客近距離觀察動物的自然行為。
- **上野農場：** 美麗的英式花園，是放鬆身心的好地方。
- **男山酒造資料館：** 了解日本清酒的釀造歷史，並品嚐優質的清酒。

## 交通資訊

從札幌搭乘特急列車前往旭川約需 1.5 小時。旭川也是前往富良野和美瑛的重要門戶。

## 溫馨提示

- 旭山動物園在冬季會有企鵝散步活動，非常受歡迎。
- 旭川拉麵以醬油湯底著稱，是來到這裡必嚐的美食。
