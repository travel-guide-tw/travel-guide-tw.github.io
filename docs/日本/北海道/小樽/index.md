# 小樽

小樽位於北海道西海岸，鄰近札幌。這座城市曾是北海道重要的金融與貿易中心，至今仍保留著許多明治與大正時期的歷史建築、石造倉庫與美麗的運河，是一座充滿浪漫氣息的海港城市。

## 主要區域

- **小樽運河區：** 標誌性的觀光區域，兩岸林立著古老的倉庫。
- **堺町通：** 繁華的商業街，聚集了玻璃工藝店、音樂盒館與各式甜點店。
- **天狗山：** 可以俯瞰小樽市區與港口全景的制高點。

## 必訪景點

- **[小樽運河](./景點/小樽運河)：** 漫步於運河邊，感受懷舊氛圍。
- **[堺町通](./景點/堺町通)：** 購買精緻的玻璃製品與品嚐北菓樓、六花亭等甜點。
- **小樽音樂盒堂：** 收藏了來自世界各地的音樂盒，門前的蒸汽鐘是著名地標。

## 交通資訊

- **火車：** 從 JR 札幌站搭乘快速列車（如快速 Airport 號）至「小樽站」，車程約 35 分鐘。
- **步行：** 小樽市區主要景點多在步行範圍內，從車站步行至運河約 10 分鐘。

## 溫馨提示

- 小樽的海鮮（特別是壽司）非常出名，建議前往「壽司屋通」品嚐。
- 傍晚時分，運河邊的瓦斯燈亮起，是拍照的最佳時刻。
