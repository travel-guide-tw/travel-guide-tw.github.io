# 天狗山 (Mt. Tengu)

天狗山是小樽市的制高點（海拔 532 公尺），以其壯麗的夜景聞名。這裡與函館山、藻岩山並稱為「北海道三大夜景」。

## 景點特色

- **天狗山纜車 (Ropeway)：** 僅需 4 分鐘即可抵達山頂。從纜車內可以俯瞰小樽港與石狩灣。
- **展望台：** 山頂設有五個不同角度的展望台，可以將小樽市區、小樽運河、甚至是遠方的積丹半島盡收眼底。
- **天狗神社與摸鼻天狗：** 山頂有一座巨大的天狗面具，據說觸摸其長鼻子可以消災解厄、實現願望。
- **冬季滑雪場：** 冬季的天狗山會轉變為滑雪場，由於坡度較陡，適合中高級滑雪者，同時也是俯瞰城市雪景的絕佳地點。

## 參觀資訊

- **門票 (纜車來回)：** 成人 1,600 日圓，兒童 800 日圓。
- **開放時間：** 09:00 - 21:00 (視季節與天氣調整)。

## 交通手段

- **巴士：** 從 JR 小樽站前的巴士總站搭乘「天狗山纜車線」巴士，至終點站下車即達纜車站。詳見 **[北海道巴士指南](../../交通/巴士)**。
