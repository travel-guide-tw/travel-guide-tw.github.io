# 北海道神宮

北海道神宮是位於北海道札幌市中央區的一座神社。北海道總鎮守。在北海道開拓初期，北海道地區多與樺太、千島等地相對望，而且為了針對俄羅斯的防守需要，因此神宮的正門是向著東北方的。神宮中的分神社「開拓神社」祭祀了一群對於北海道開拓事業有功勞的人物（如島義勇、間宮林藏）。
