# 旭山動物園

旭山動物園內主要為寒帶動物，園方精心打造最貼近自然棲息地的環境，為園內種類超過 124 種的 700 多隻動物提供舒適的環境。

- 地址 〒078-8205 北海道旭川市東旭川町倉沼
- 電話號碼 +81166361104
- 交通手段 從 JR 旭川駅東口前第六號月台搭乘旭川電氣軌道巴士（旭山動物園線 41、42、47 號巴士）。
- 營業時間 9:30 ～ 17:15
