# 薄野 (Susukino)

薄野是札幌最大的娛樂區，也是日本三大紅燈區之一（與東京歌舞伎町、福岡中洲並列）。這裡聚集了數千家餐廳、酒吧、卡拉 OK 與夜總會，是札幌夜生活的核心。

## 景點特色

- **霓虹燈海：** 著名的 NIKKA 威士忌看板所在地。入夜後，巨大的廣告牌與無數店家的霓虹燈將薄野妝點成不夜城。
- **拉麵橫丁 (Ramen Yokocho)：** 位於薄野中心的小巷，匯集了多家歷史悠久的札幌拉麵店，是品嚐正宗札幌味噌拉麵的聖地。
- **螃蟹料理：** 區域內有多家著名的螃蟹專賣店。
- **冬季活動：** 札幌雪祭期間，薄野會場會展示精美的「冰雕」。

## 溫馨提示

- 薄野地區夜晚非常熱鬧且相對安全，但仍建議避開過於主動招攬生意的路人。
- 許多餐廳營業至深夜或凌晨，適合看完夜景後來此宵夜。
