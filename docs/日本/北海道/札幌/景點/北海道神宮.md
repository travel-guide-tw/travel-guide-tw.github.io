# 北海道神宮

北海道神宮是北海道總鎮守（地位最高的神社），位於圓山公園旁。這裡祭祀著開拓北海道的守護神，是當地人祈求平安、學業進步與良緣的重要場所。

## 景點特色

- **開拓神社：** 祭祀對北海道開拓有功的人士。
- **正門朝向：** 神宮的正門面向東北方，這在當時是為了防範來自北方的威脅。
- **櫻花與梅花：** 春季時，這裡可以同時欣賞到櫻花與梅花盛開，是札幌著名的賞花勝地。
- **六花亭神宮茶屋：** 位於境內，可以品嚐到現烤的「判官樣」餅（神宮限定甜點）。

## 參觀資訊

- **地址：** 〒064-8505 北海道札幌市中央區宮丘 474
- **電話：** +81 11-611-0261
- **開放時間：**
  - 夏季：約 06:00 - 17:00
  - 冬季：約 07:00 - 16:00
  - (具體開門時間依季節變動，請參考官網)
- **門票：** 免費參觀。

## 交通手段

- **地鐵：** 搭乘札幌市營地下鐵東西線至「圓山公園站」，步行約 15 分鐘。
- **巴士：** 搭乘 JR 巴士（西 14、西 15）至「神宮前」站下車。
