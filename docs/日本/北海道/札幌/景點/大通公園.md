# 大通公園

大通公園位於札幌市中心，橫跨東西約 1.5 公里，是札幌的象徵，也是市民休憩的場所。

[大通公園官方相片藝廊](https://odori-park.jp/miryoku/photo-gallery/)
_圖片來源：公益財團法人札幌市公園綠化協會_

## 景點特色

大通公園將札幌市中心分為南北兩部分。公園內種滿了花草樹木，四季景色分明。這裡也是舉辦各種大型活動的場地，最著名的莫過於冬季的「札幌雪祭」。

## 主要活動

- **[札幌雪祭](../../_活動/札幌雪祭.md) (2 月)：** 展示巨大的冰雕與雪雕，是大通公園年度最盛大的活動。
- **丁香節 (5 月)：** 慶祝初夏到來的祭典。
- **[啤酒花園](../../_活動/index.md#夏季-7月---8月) (7-8 月)：** 日本規模最大的戶外啤酒花園。
- **白色燈樹節 (11-12 月)：** 璀璨的燈飾裝點冬夜。

## 門票資訊

- **公園進入：** 免費。
- **札幌電視塔 (展望台)：** 成人 1,000 日圓。

## 交通手段

- **地鐵：** 搭乘南北線、東西線或東豐線至「大通站」下車即達。詳見 **[札幌地鐵指南](../../_交通/鐵路與地鐵.md)**。
- **步行：** 從 JR 札幌站步行約 10-15 分鐘。

## 溫馨提示

- 公園內的「烤玉米」攤位是札幌的特色美食，推薦嘗試。
- 札幌電視塔位於公園的最東端（1 丁目），是俯瞰公園全景的最佳位置。
