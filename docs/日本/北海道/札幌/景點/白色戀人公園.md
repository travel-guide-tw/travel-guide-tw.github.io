# 白色戀人公園 (Shiroi Koibito Park)

白色戀人公園是由北海道著名甜點「白色戀人」的製造商——石屋製菓 (ISHIYA) 所營運的主題公園。這裡結合了巧克力工廠參觀、甜點製作體驗與如歐洲莊園般的優美庭園。

## 景點特色

- **工廠參觀：** 遊客可以透過玻璃窗觀察著名的「白色戀人」夾心餅乾的自動化生產線。
- **甜點體驗：** 提供親子同樂的甜點教室，可以親手製作心形的大型白色戀人餅乾。
- **歐式庭園：** 園內有玫瑰花園、都鐸式建築以及每小時鳴響一次的機械時鐘塔，是非常熱門的拍照打卡點。
- **冬季點燈：** 冬季夜晚，庭園會布置璀璨的燈飾，營造出如童話般的氛圍。

## 參觀資訊

- **地址：** 〒063-0052 北海道札幌市西區宮之澤 2 條 2 丁目
- **門票：** 庭園免費進入。工廠參觀（收費區）成人約 800 - 1,000 日圓。
- **開放時間：** 10:00 - 18:00 (最後入場 17:00)。

## 交通手段

- **地鐵：** 搭乘東西線至「宮之澤站」下車，步行約 7 分鐘。詳見 **[札幌地鐵指南](../../交通/鐵路與地鐵)**。
