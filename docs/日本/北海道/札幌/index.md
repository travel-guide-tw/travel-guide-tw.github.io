# 札幌

## 北海道的首府

札幌是北海道最大的城市，也是其行政、經濟和文化中心。這座城市以其開闊的街道、美麗的公園以及冬季的雪祭而聞名。

## 必訪景點

- **[北海道大學](./景點/北海道大學)：** 日本著名的國立大學，校園內綠意盎然，秋季的銀杏大道更是美不勝收。
- **[北海道神宮](./景點/北海道神宮)：** 北海道總鎮守，是當地人祈求平安與好運的重要場所。
- **[大通公園](./景點/大通公園)：** 位於市中心，是舉辦札幌雪祭的主要場地。

## 交通資訊

札幌市內交通發達，擁有完善的地鐵、巴士和路面電車系統。從新千歲機場搭乘快速列車只需約 40 分鐘即可抵達札幌車站。

## 溫馨提示

- 札幌雪祭通常在每年 2 月舉行，建議提前預訂住宿。
- 札幌以其美味的拉麵、湯咖哩和成吉思汗烤肉而聞名，千萬不要錯過。
