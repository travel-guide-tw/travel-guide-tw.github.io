# 函館

## 浪漫的港口城市

函館位於北海道南端，是日本最早對外開放的港口之一。這裡融合了西方與日本的文化，擁有迷人的夜景、歷史悠久的建築和新鮮的海產。

## 必訪景點

- **[函館山](./景點/函館山)：** 這裡的夜景被譽為「世界三大夜景」之一，是遊客必訪的地點。
- **[函館朝市](./景點/函館朝市)：** 可以在這裡品嚐到最新鮮的海鮮，體驗熱鬧的市場氛圍。
- **[金森紅磚倉庫](./景點/金森紅磚倉庫)：** 由歷史倉庫改建的購物與餐飲區，充滿懷舊氛圍。
- **[元町教會群](./景點/元町教會群)：** 充滿異國風情的教會建築與坡道，見證了函館的開港歷史。
- **[大沼國定公園](./景點/大沼國定公園)：** 位於函館近郊，可以欣賞到壯麗的駒之岳和美麗的湖泊。
- **[五稜郭公園](./景點/五稜郭公園)：** 獨特的星形要塞，是賞櫻和了解歷史的好去處。

## 交通資訊

從札幌搭乘特急列車前往函館約需 3.5 至 4 小時。函館市內可以利用路面電車（路面電車）輕鬆遊覽主要景點。

## 溫馨提示

- 函館山的夜景受天氣影響較大，建議在晴朗的傍晚前往。
- **美食推薦：**
  - **きくよ食堂：** 函館朝市內的老字號，以炭火烘烤的海鮮丼聞名。
  - **阿佐利 本店：** 位於元町地區的百年壽喜燒老店。
- 函館的鹽味拉麵是當地的特色美食，值得一試。
