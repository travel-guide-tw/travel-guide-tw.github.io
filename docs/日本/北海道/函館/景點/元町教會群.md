# 元町教會群

函館元町地區位於函館山腳下，是函館開港初期外國人居留地的中心。這裡保留了許多充滿異國風情的坡道與歷史建築，其中最著名的便是風格各異的教會群。

![元町教會群](https://upload.wikimedia.org/wikipedia/commons/d/d7/Motomachi_Catholic_Church_in_Hakodate_Hokkaido_Japan03n.jpg)
_圖片來源：Wikimedia Commons_

## 景點特色

- **函館哈利斯特斯東正教會：** 日本最古老的東正教會，以其白牆與綠色屋頂的拜占庭風格建築聞名，被譽為「叮噹寺」（因為鐘聲動聽）。
- **元町天主教會：** 哥德式建築，擁有高聳的尖塔。內部的祭壇是羅馬教皇贈送的，非常珍貴。
- **函館聖約翰教會：** 現代風格的建築，從空中俯瞰呈現十字架形狀，設計獨特。
- **異國坡道：** 教會群周邊有許多著名的坡道（如八幡坂），可以從高處俯瞰函館港，是熱門的攝影地點。

## 參觀資訊

- **地址：** 〒040-0054 北海道函館市元町
- **門票：** 教堂外部參觀免費，部分教堂內部參觀需付費（約 200-300 日圓）。
- **開放時間：** 各教堂開放時間不一，通常為 10:00 - 16:00。

## 交通手段

- **路面電車：** 搭乘函館市電至「末廣町」或「十字街」站下車，步行約 10-15 分鐘（需爬坡）。

## 溫馨提示

- 教堂內部為神聖場所，參觀時請保持安靜，並遵守禁止攝影的規定。
- 建議穿著好走的鞋子，因為元町地區多為陡峭的坡道。
