# 富良野薰衣草節

夏季的富良野以其紫色的大地聞名於世，是北海道最浪漫的旅遊時段。

## 最佳賞期

- **花季**：6 月下旬至 8 月上旬。
- **盛開期**：通常在 **7 月中旬至 7 月下旬**，此時薰衣草顏色最為鮮艷濃郁。

## 推薦觀賞地點

### 富田農場 (Farm Tomita)

富良野最著名的農場，也是薰衣草旅遊的發源地。

- **特色**：擁有壯觀的「彩色花田」以及不同品種的薰衣草田。
- **必吃**：薰衣草口味霜淇淋、薰衣草汽水。

### 北星山町營薰衣草園

位於中富良野，可以搭乘觀光吊椅上山俯瞰紫色山坡。

- **特色**：視野極佳，可以同時看到薰衣草田與十勝岳連峰。

### 上富良野日之出公園

擁有 360 度的展望台。

- **特色**：公園內的白色鐘塔是熱門的婚紗與拍照景點。

## 交通建議

- **薰衣草臨時列車 (Norokko)**：夏季限定，往返於旭川、美瑛與富良野之間，會停靠臨時站「薰衣草花田站」。詳見 **[JR 普通與快速列車指南](../_交通/鐵路與地鐵.md)**。
- **租車自駕**：在富良野與美瑛地區自駕最為便利。詳見 **[北海道租車自駕指南](../_交通/租車.md)**。
