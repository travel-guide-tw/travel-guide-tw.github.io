# 札幌雪祭 (Sapporo Snow Festival)

札幌雪祭是北海道冬季規模最大、最著名的活動，每年吸引超過 200 萬名遊客前往。

## 活動概要

- **日期**：每年 2 月上旬（通常為期 7-8 天）。
- **地點**：札幌市區（大通公園、薄野、TSUDOME）。

## 主要會場

### 大通會場 (Odori Site)

位於札幌市中心的 **[大通公園](../札幌/景點/大通公園)**，是雪祭的主戰場。

- **特色**：展示巨大的雪雕作品，主題涵蓋世界著名建築、動漫人物或時事。
- **夜間點燈**：入夜後雪雕會搭配彩色燈光與投射秀 (Projection Mapping)，非常壯觀。

### 薄野會場 (Susukino Site)

位於著名的薄野娛樂區。

- **特色**：以「冰雕」為主，展示精緻透明的冰雕作品。
- **互動**：部分冰雕內嵌有鮮魚或其他裝飾品。

### TSUDOME 會場

適合親子同樂的郊區會場。

- **特色**：提供巨大的滑雪道、雪地橡皮艇等體驗設施。

## 旅遊建議

1. **服裝**：2 月的札幌非常寒冷，且路面易滑，建議穿著防水保暖的鞋子並搭配防滑墊。
2. **住宿**：雪祭期間札幌市內飯店往往一票難求，建議提前 3-6 個月預約。
3. **飲食**：大通會場設有許多美食攤位，可以品嚐到北海道各地的熱食。
