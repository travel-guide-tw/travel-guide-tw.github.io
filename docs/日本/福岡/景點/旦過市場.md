# 旦過市場

## 基本資訊
- **地址**：福岡縣北九州市小倉北區魚町 4-2-1
- **營業時間**：08:00-18:00（各店鋪略有不同）
- **歷史**：創立於大正時期，九州最古老市場之一

## 特色體驗
- 「大學堂」食堂：自選新鮮食材現場烹調
- 在地美食：關門海峽直送海鮮、小倉名物「燒咖哩」
- 每月第三個週日舉辦「古董市集」

## 交通方式
- JR 小倉站步行約 8 分鐘
- 北九州單軌電車「旦過站」直結
- 市場專用停車場（前 2 小時免費）
