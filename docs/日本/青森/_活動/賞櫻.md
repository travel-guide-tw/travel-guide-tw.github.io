# 青森賞櫻指南 (弘前櫻花祭)

青森縣擁有日本首屈一指的賞櫻勝地，其中以「弘前櫻花祭」最為著名。

## 弘前櫻花祭 (Hirosaki Cherry Blossom Festival)

位於 **[弘前城](../弘前/景點/弘前城)** (弘前公園)，被公認為日本最美的櫻花祭之一。

- **日期**：通常在 4 月下旬至 5 月上旬。
- **亮點**：
  - **櫻花地毯 (Hanaikada)**：外護城河的水面上鋪滿飄落的櫻花瓣，形成粉紅色的地毯。
  - **櫻花隧道**：西護城河兩岸的櫻花交織成隧道。
  - **夜間點燈**：入夜後燈光投射在古城與櫻花上，展現出極致的幻境美。

## 交通建議

- **地點**：從 **[JR 弘前車站](../弘前/)** 搭乘「100 日圓循環巴士」至「市役所前」下車即可到達弘前公園入口。詳見 **[弘前交通指南](../交通/巴士)**。

## 其他賞櫻名所

- **蘆野公園**：位於五所川原市，火車穿過櫻花隧道的景致是熱門攝影點。
- **十和田市官廳街通**：擁有 150 棵櫻花樹的絕美大道。
