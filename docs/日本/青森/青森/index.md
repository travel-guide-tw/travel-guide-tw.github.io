# 青森市 (Aomori City)

青森市是青森縣的行政、經濟與文化中心，也是前往北海道的門戶。

## 必訪景點

- **[睡魔之家 Wa Rasse](./景點/睡魔之家Warasse)**：展示祭典中實際使用過的巨大燈籠。
- **[八甲田山](./景點/八甲田山)**：搭乘纜車欣賞四季美景，冬季有樹冰。
- **[酸湯溫泉](./景點/酸湯溫泉)**：擁有 300 年歷史的古老溫泉，以「千人風呂」聞名。
- **古川市場 (青森魚菜中心)**：體驗自選海鮮丼 (Nokke-don)。
- **三內丸山遺跡**：日本規模最大的繩文時代聚落遺跡（世界文化遺產）。

## 活動與祭典

- **[青森睡魔祭](../活動/睡魔祭)**：每年 8 月舉行。
