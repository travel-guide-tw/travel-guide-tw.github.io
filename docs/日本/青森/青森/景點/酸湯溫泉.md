# 酸湯溫泉

酸湯溫泉位於八甲田山麓，海拔約 900 公尺，是一座擁有超過 300 年歷史的古老溫泉。它以其巨大的「千人風呂」和強酸性的泉質而聞名，是日本著名的國民保養溫泉地。

![酸湯溫泉](https://upload.wikimedia.org/wikipedia/commons/4/40/Sukayu_Onsen_01.jpg)
_圖片來源：Wikimedia Commons_

## 景點特色

- **千人風呂：** 溫泉旅館內著名的巨大木造浴場，面積約 248 平方公尺，全部由檜木製成，氣氛莊嚴且充滿懷舊感。
- **強酸性泉質：** 泉水含有豐富的酸性硫磺成分，對神經痛、風濕、皮膚病等有顯著療效。
- **豪雪地帶：** 這裡曾創下日本氣象觀測史上最高的積雪紀錄（566 公分），冬季時整座旅館會被厚厚的白雪覆蓋，景色壯觀。
- **溫泉療養 (湯治)：** 這裡至今仍保留著傳統的「湯治」文化，提供長期住宿的療養方案。

## 參觀資訊

- **地址：** 〒030-0111 青森縣青森市荒川南荒川山國有林酸湯澤 50
- **電話：** +81 17-738-6400
- **日歸入浴時間：** 07:00 - 17:30 (千人風呂)
- **門票 (日歸)：** 成人 1,000 日圓，兒童 500 日圓。

## 交通手段

- **巴士：** 從 JR 青森站或新青森站搭乘 JR 巴士（往十和田湖方向）至「酸湯溫泉」站下車，車程約 70-80 分鐘。詳見 **[青森巴士指南](../../_交通/巴士.md)**。
- **鐵路：** 最近的主要車站為 **[JR 青森站](../index.md)** 或 **[新青森站](../../青森/index.md)**。詳見 **[青森鐵路指南](../../_交通/鐵路.md)**。

## 溫馨提示

- **千人風呂為混浴：** 雖然是混浴，但設有女性專用時段（通常為 08:00-09:00 及 20:00-21:00），且女性可穿著專用的入浴衣。
- 由於泉水酸性極強，泡湯時請注意保護眼睛，且不建議將銀飾帶入池中（會變黑）。
