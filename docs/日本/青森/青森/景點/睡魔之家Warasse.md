# 睡魔之家 Wa Rasse (Nebuta House Warasse)

睡魔之家 Wa Rasse 位於青森車站旁，是一座介紹青森睡魔祭歷史與魅力的館覽設施。即使不是在 8 月祭典期間，遊客也能在此近距離欣賞巨大的睡魔燈籠。

## 景點特色

- **巨大燈籠展示：** 館內展示了祭典中實際使用過、且獲得大獎的數座巨大睡魔燈籠，震撼力十足。
- **祭典體驗：** 遊客可以親自嘗試敲擊祭典太鼓，或在特定時段與工作人員一起練習跳人 (Haneto) 的舞步。
- **建築設計：** 建築外觀由深紅色的鋼條包圍，設計獨特且富有現代感，已成為青森港口的地標。
- **紀念品商店：** 販售豐富的睡魔祭周邊商品與青森在地特產。
