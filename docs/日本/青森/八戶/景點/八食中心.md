# 八食中心 (Hasshoku Center)

八食中心位於青森縣八戶市，是一座大型的室內海鮮市場。這裡匯集了從八戶港及周邊海域捕獲的各式新鮮海產、乾貨、在地日本酒以及青森名產。

## 景點特色

- **新鮮海產：** 市場內有多達 60 多家店舖，販售剛捕獲的鮮魚、螃蟹、海膽與干貝。
- **七厘村 (Shichirin-mura)：** 這是八食中心最具特色的體驗。遊客可以在市場內購買喜歡的海鮮或肉類，然後租借炭火烤爐（七厘），在專用區域現場燒烤享用。
- **廚房體育館：** 設有美食街，提供海鮮丼、壽司以及八戶特色料理「仙貝湯」(Senbei-jiru)。
- **伴手禮首選：** 這裡也是購買青森蘋果相關產品與各類海產加工品的絕佳地點。

## 參觀資訊

- **地址：** 〒039-1161 青森縣八戶市河原木字神才 22-2
- **門票：** 免費進入（七厘村租借及用餐另計）。
- **開放時間：**
  - 市場：09:00 - 18:00
  - 飲食區：09:00 - 19:00 (週三通常為休館日，建議出發前確認)
