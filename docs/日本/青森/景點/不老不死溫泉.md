# 不老不死溫泉

不老不死溫泉位於青森縣西津輕郡深浦町，以其緊鄰日本海的露天風呂而聞名。溫泉水含有豐富的鐵質，呈現獨特的紅褐色。

![不老不死溫泉](https://upload.wikimedia.org/wikipedia/commons/9/9b/Furofushi_Onsen.jpg)
_圖片來源：Wikimedia Commons_

## 景點特色

- **海邊露天風呂：** 溫泉池就設在海岸邊的岩石上，與海面幾乎齊平。泡湯時可以聽著浪濤聲，欣賞壯觀的日本海日落。
- **紅褐色泉質：** 泉水富含鐵質與鹽分，據說對神經痛、皮膚病等有療效，且能讓身體持久保溫。
- **日落絕景：** 這裡被譽為日本觀賞日落最美的溫泉之一，傍晚時分金色的陽光灑在海面上，景色極其夢幻。

## 參觀資訊

- **地址：** 〒038-2324 青森縣西津輕郡深浦町大字舮作字下清瀧 15
- **電話：** +81 173-74-3500
- **日歸入浴時間：** 08:00 - 16:00 (最後入場 15:30)
- **門票 (日歸)：** 成人 700 日圓，兒童 400 日圓。

## 交通手段

- **火車：** 搭乘 JR 五能線至「WeSPa 椿山站」，旅館提供免費接駁巴士（需配合火車到站時間）。
- **自駕：** 從青森市區出發約需 2.5 至 3 小時。

## 溫馨提示

- 露天風呂為混浴（有女性專用時段），且為了保護景觀與環境，露天池禁止使用肥皂或洗髮精。
- 由於緊鄰海邊，若遇強風或大浪，露天風呂可能會暫時關閉。
