# 弘前 (Hirosaki)

弘前是津輕地區的核心城市，保留了深厚的城下町文化與歷史建築。

## 必訪景點

- **[弘前城 (弘前公園)](./景點/弘前城)**：日本現存十二天守之一，著名的賞櫻勝地。
- **星巴克弘前公園前店**：由登錄有形文化財的古建築改建。
- **津輕藩睡魔村**：體驗津輕文化、手工藝與參觀扇形燈籠。
- **最勝院五重塔**：東北地區最美的五重塔。

## 活動與祭典

- **[弘前櫻花祭](../活動/賞櫻)**：春季限定。
- **弘前睡魔祭 (Neputa)**：夏季限定。

## 交通資訊

- **[JR 弘前站]**：從新青森搭乘特急約 30 分鐘。
- **[弘南巴士](../交通/巴士)**：市區內最方便的移動工具。
