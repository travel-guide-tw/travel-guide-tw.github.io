# 青森鐵路交通指南

青森縣的鐵路網路包含新幹線、JR 在來線、私鐵以及獨特的觀光列車。

## 東北新幹線

連接東京、仙台與青森的大動脈。

- **起訖點：東京 ↔ 新青森**
- **車次**：Hayabusa (隼) 號為全車指定席，最快約 3 小時抵達。
- **主要停靠站**：[八戶站](../八戶/)、七戶十和田站、**[新青森站](../青森/)**。

## JR 在來線

- **JR 奧羽本線**：
  - **起訖點：新青森 ↔ 弘前 ↔ 大館**
  - 連接青森市與 **[弘前](../弘前/)** 的主要路線。
- **JR 五能線**：
  - **起訖點：弘前 ↔ 深浦 ↔ 東能代**
  - 日本最著名的景觀鐵路之一，著名的觀光列車 **[休閒白神號 (Resort Shirakami)](https://www.jreast.co.jp/multi/zh-CHT/joyful/shirakami.html)** 運行於此。

## 私鐵與第三部門鐵路

- **青森鐵道 (Aoimori Railway)**：
  - **起訖點：青森 ↔ 八戶 ↔ 目時**
  - 承接原東北本線的路線，是青森與 **[八戶](../八戶/)** 之間的重要連接。
- **津輕鐵路**：
  - **起訖點：津輕五所川原 ↔ 津輕中里**
  - 以冬季的 **[暖爐列車 (Stove Train)](https://tsutetsu.com/stove.html)** 聞名。
