# 青森渡輪交通指南

青森是本州與北海道之間海運的樞紐，提供多條航線，特別適合自駕或是想要體驗船旅的遊客。

## 青函航線 (青森 ↔ 函館)

這是最繁忙的航線，連接青森市與北海道函館市。

- **起訖點：青森港 ↔ [函館港](../../北海道/_交通/渡輪.md)**
- **營運公司**：
  - **[津輕海峽渡輪 (Tsugaru Kaikyo Ferry)](https://www.tsugarukaikyo.co.jp/)**：船隻較新且大，設有舒適的休息區與寵物友善空間。
  - **[青函渡輪 (Seikan Ferry)](https://www.seikan-ferry.co.jp/)**：價格較為實惠，班次密集。
- **所需時間**：約 3 小時 40 分鐘。
- **特色**：24 小時皆有班次，可載運自用小客車。

## 大函航線 (大間 ↔ 函館)

這是連接本州最北端「大間町」與函館的最短航線。

- **起訖點：大間港 ↔ [函館港](../../北海道/_交通/渡輪.md)**
- **營運公司**：[津輕海峽渡輪](https://www.tsugarukaikyo.co.jp/)
- **所需時間**：約 90 分鐘。
- **特色**：航程最短，是從下北半島前往北海道最快的方式。

## 相關景點

- **[函館](../../北海道/函館/index.md)**：抵達北海道後的首站。
- **大間町**：以「大間鮪魚」聞名。
