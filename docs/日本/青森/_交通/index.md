# 青森交通指南

青森縣位於東北地區最北端，是連接本州與北海道的重要樞紐。主要的交通方式包括東北新幹線、青森機場、以及發達的巴士網路。

## 如何抵達青森

### 東北新幹線

東北新幹線是前往青森最快速的方式。

- **[新青森站](../青森/)**：新幹線終點站，可在此轉乘 JR 在來線前往青森市區或弘前。
- **[八戶站](../八戶/)**：前往十和田湖或下北半島的重要門戶。
- **[七戶十和田站]**：鄰近十和田市。

### 飛機 (Airplane)

- **青森機場 (AOJ)**：位於青森市區郊外，有往返東京（羽田）、大阪（伊丹）、札幌（新千歲）及名古屋的航班。
- **三澤機場 (MSJ)**：位於三澤市，適合前往八戶或下北半島。

## 青森境內交通

### 鐵路 (JR & 私鐵)

- **JR 奧羽本線**：連接青森、弘前與大館。
- **青森鐵道 (Aoimori Railway)**：連接青森、野邊地與八戶。
- **津輕鐵路**：以冬季的「暖爐列車」聞名。

### 巴士 (Bus)

巴士是前往奧入瀨溪流、十和田湖及酸湯溫泉等山區景點的主要交通工具。

- **JR 巴士東北**：營運前往十和田湖與八甲田山的路線。

### 租車自駕 (Rental Car)

青森縣面積廣大，特別是前往下北半島或深浦地區，自駕是最為靈活的選擇。

## 往返北海道 (To & From Hokkaido)

青森是連接本州與北海道的關鍵點，有以下兩種主要方式：

- **北海道新幹線**：
  - **起訖點：新青森 ↔ [新函館北斗](../../北海道/_交通/鐵路與地鐵)**
  - 穿過青函隧道，僅需約 1 小時即可抵達北海道。
- **渡輪 (Ferry)**：
  - **起訖點：青森港 / 大間港 ↔ [函館港](../../北海道/_交通/渡輪)**
  - 適合自駕旅客將車輛運往北海道，或想體驗海上航行的遊客。詳見 **[青森渡輪指南](./渡輪)**。

## 核心交通網絡 (Major Connections)

| 起訖點 (Point A ↔ Point B) | 推薦交通工具            | 大約時間     |
| :-------------------------- | :---------------------- | :----------- |
| **新青森 ↔ 青森車站**      | JR 奧羽本線 / 巴士      | 5-15 分鐘    |
| **青森 ↔ 弘前**            | JR 奧羽本線 (特急/快速) | 30-45 分鐘   |
| **青森 ↔ 八戶**            | 青森鐵道                | 1.5 小時     |
| **青森 ↔ 八甲田山/酸湯**   | JR 巴士                 | 1 - 1.2 小時 |
| **青森/八戶 ↔ 十和田湖**   | JR 巴士 (觀光路線)      | 2.5 - 3 小時 |
| **弘前 ↔ 深浦**            | JR 五能線 (休閒白神號)  | 2 - 2.5 小時 |
