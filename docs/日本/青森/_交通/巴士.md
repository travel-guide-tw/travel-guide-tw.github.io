# 青森巴士交通指南

巴士是前往青森縣山區、湖泊與偏遠地區的重要交通工具。

## 觀光與長途巴士 (JR 巴士東北)

JR 巴士東北營運多條連接主要車站與自然景點的路線。

### 奧入瀨號 / 十和田湖號

- **起訖點：[青森車站](../青森/) / [新青森](../青森/) / [八戶](../八戶/) ↔ [奧入瀨溪流](../十和田/景點/奧入瀨溪流) ↔ [十和田湖](../十和田/景點/十和田湖)**
- **特色**：前往奧入瀨與十和田湖最方便的大眾運輸。冬季部分路段停駛或縮短路線。
- **時刻表與詳情**：[JR 巴士東北官網](https://www.jrbustohoku.co.jp/towada_guide/tw/)

### 八甲田線

- **起訖點：[青森車站](../青森/) ↔ [八甲田山](../青森/景點/八甲田山) ↔ [酸湯溫泉](../青森/景點/酸湯溫泉)**
- **特色**：前往八甲田纜車與酸湯溫泉的必經路線。
- **時刻表**：[青森 - 八甲田時刻表](https://www.jrbustohoku.co.jp/citic_route/aomori-hakkoda/)

## 市區公車

- **青森市營巴士**：服務青森市中心及周邊區域。
- **弘南巴士**：服務 **[弘前](../弘前/)** 市區，包括著名的「100 日圓循環巴士」。
