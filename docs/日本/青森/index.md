# 青森

## 蘋果與祭典之鄉

青森縣位於本州島的最北端，以其美味的蘋果、壯觀的睡魔祭以及豐富的自然景觀而聞名。從壯麗的奧入瀨溪流到歷史悠久的弘前城，青森為遊客提供了獨特的旅遊體驗。

## 必訪景點

- **[奧入瀨溪流](./景點/奧入瀨溪流)：** 日本最美的溪流之一，沿途有許多瀑布和奇岩怪石，是健行和攝影的絕佳地點。
- **[不老不死溫泉](./景點/不老不死溫泉)：** 緊鄰日本海的露天風呂，以紅褐色的泉質與壯觀的日落絕景聞名。
- **[弘前城](./景點/弘前城)：** 現存十二天守之一，是日本著名的賞櫻勝地，冬季的雪燈籠祭也非常迷人。
- **[八甲田山](./景點/八甲田山)：** 搭乘纜車欣賞壯麗的火山連峰，冬季可觀賞巨大的「樹冰」奇景。
- **[酸湯溫泉](./景點/酸湯溫泉)：** 擁有 300 年歷史的古老溫泉，以巨大的木造「千人風呂」聞名。
- **[十和田湖](./景點/十和田湖)**：跨越青森和秋田兩縣的火山口湖，以其清澈的湖水和四季分明的景色而聞名。

## 交通資訊

前往青森可以搭乘東北新幹線至新青森站，或是搭乘飛機至青森機場。在縣內觀光，可以利用 JR 鐵路、巴士或租車等交通工具。

## 溫馨提示

- 青森睡魔祭在每年 8 月舉行，是日本最具代表性的祭典之一，建議提前預訂住宿。
- **美食推薦：**
  - **古川市場 (青森魚菜中心)：** 著名的「自選海鮮丼」(のっけ丼)，可以挑選自己喜歡的配料。
  - **味の札幌 大西：** 青森特色「味噌咖哩牛奶拉麵」的創始店。
  - **三忠食堂：** 弘前市著名的老字號食堂，以津輕蕎麥麵聞名。
  - **居酒屋さくら：** 弘前地區受歡迎的在地居酒屋。
- 青森的蘋果產量位居日本第一，不妨品嚐一下當地的蘋果汁和甜點。
