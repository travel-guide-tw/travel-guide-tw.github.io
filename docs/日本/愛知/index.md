# 愛知

## 武將之鄉與工業中心

愛知縣位於日本中部地區，是戰國三傑（織道信長、豐臣秀吉、德川家康）的故鄉，也是日本現代工業的重要基地。

## 主要區域與景點

- **[名古屋](./)：** 愛知縣的首府，以名古屋城及獨特的飲食文化（名古屋飯）聞名。
- **[犬山](./)：** 擁有國寶犬山城。
- **[吉卜力公園](./)：** 位於長久手市，是吉卜力工作室作品的主題公園。

## 交通資訊

- **[交通概覽](./交通/)：** 包含名古屋市內交通及中部國際機場連接資訊。

## 溫馨提示

- 名古屋市內的地下鐵非常便利，建議購買地下鐵一日券。
- 鰻魚飯（Hitsumabushi）是愛知縣必嚐的特色美食。
