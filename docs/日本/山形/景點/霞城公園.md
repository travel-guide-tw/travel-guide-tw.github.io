# 霞城公園

山形城舊址（霞城公園）及庭院現已成為一座廣闊的公共公園，緬懷著曾經統治這裡的強盛最上氏大名。這裡也是漫步、野餐和春季賞櫻的熱門場所。

- 地址 〒990-0826 山形県山形市霞城町 1−7
- 電話號碼 +81236411212
- 交通手段 山形城坐落於霞城公園附近，從山形站步行 15 分鐘，或乘坐一小段開往天童站的巴士即可抵達，請在霞城公園前站（第二站）下車，車程約 3 分鐘，隨後步行 5 分鐘即可抵達公園中心。
- 營業時間 5:00 ～ 22:00
