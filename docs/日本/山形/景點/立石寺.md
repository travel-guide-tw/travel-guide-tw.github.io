# 立石寺

立石寺是位在日本山形縣山形市之天台宗寺院。以山寺之名聞名。內有日本重要文化財：立石寺中堂、立石寺三重小塔、木造藥師如來坐像、木造慈覺大師頭部・木棺、天養元年如法經所碑。

- 地址 〒999-3301 山形県山形市山寺 4456-1
- 電話號碼 +81236952002
- 交通手段 山寺的下層寺廟距 JR 山寺站步行僅需七分鐘。
- 營業時間 8:00 ～ 16:00
