# 鳥海山

「鳥海山」是「日本百大名山」之一，從日本海海抜 0 公尺至標高 2,236 公尺處傲然矗立，十分罕見。山姿高雅，與富士山類似，又名「出羽富士」。

- 地址 山形縣飽海郡遊佐町吹浦
- 交通手段 遊佐站至吹浦口（大平）開車約 40 分）
