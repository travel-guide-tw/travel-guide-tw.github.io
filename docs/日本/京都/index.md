# 京都

## 千年古都

京都是日本最古老的城市之一，於 794 年被選為日本天皇的所在地，擁有豐富的歷史文化遺產。這座城市以其傳統的寺廟、神社、庭園和藝伎區而聞名，吸引著來自世界各地的遊客。

## 必訪景點

- **[清水寺](./景點/清水寺)：** 懸空的清水舞台。
- **[金閣寺](./景點/金閣寺)：** 金碧輝煌的舍利殿。
- **[伏見稻荷大社](./景點/伏見稻荷大社)：** 壯觀的千本鳥居。

## 交通資訊

- **[交通概覽](./交通/)：** 前往京都市可以搭乘新幹線至京都車站。從京都車站可以搭乘 JR、地鐵或巴士前往市內各個景點。

## 鄰近區域推薦

從京都出發，您可以輕鬆前往關西其他著名城市：
- **[大阪府](../大阪/)：** 享受豐富美食與繁華街景。
- **[奈良縣](../奈良/)：** 距離京都僅一小時車程，適合一日遊。
- **[滋賀縣](../滋賀/)：** 探訪日本第一大湖「琵琶湖」，欣賞絕美自然風光。

## 溫馨提示

- 京都市的交通非常便利，建議購買巴士一日券或地鐵一日券，以便更輕鬆地遊覽市區景點。
- 京都市的寺廟和神社眾多，建議事先規劃好行程，以便更好地體驗當地的歷史文化。
- 京都市的美食種類繁多，從懷石料理到拉麵，應有盡有，滿足您的味蕾。
