# 平等院

平等院位於京都府宇治市，始建於 1052 年。其最著名的建築是「鳳凰堂」，堂內供奉著阿彌陀如來坐像，展現了平安時代極樂淨土的景象。

- 地址：〒 611-0021 京都府宇治市宇治蓮華 116
