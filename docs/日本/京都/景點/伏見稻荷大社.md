# 伏見稻荷大社

伏見稻荷大社位於稻荷山的山腳，是日本全國約 3 萬座稻荷神社的總本宮。這裡以「千本鳥居」聞名於世，成千上萬座朱紅色的鳥居漫延在山間小徑上，形成了一條壯觀的隧道。這裡是遊客祈求生意興隆、五穀豐登的重要神聖之地。

- 地址：〒 612-0882 京都市伏見區深草藪之內町 68
