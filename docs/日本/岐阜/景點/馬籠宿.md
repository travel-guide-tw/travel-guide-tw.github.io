# 馬籠宿

![馬籠宿](https://upload.wikimedia.org/wikipedia/commons/f/f0/Magome-Juku_StoneSlope.jpg)
_圖片來源：Wikimedia Commons_

馬籠宿位於岐阜縣中津川市，是中山道 69 次中的第 43 號宿場。這裡與長野縣的**[妻籠宿](../../長野/景點/妻籠宿)**齊名，兩地之間的「妻籠－馬籠古道」是熱門的健行路線。

## 景點特色

- **坡道驛站：** 馬籠宿建立在險峻的山坡上，這在全日本的宿場中非常罕見。
- **石疊小徑：** 整條老街鋪設著石板路，兩旁木造建築錯落有致。
- **中山道健行：** 往返於馬籠宿與**[妻籠宿](../../長野/景點/妻籠宿)**之間，步行約 2.5 至 3 小時，是感受古日本氛圍的最佳體驗。
- **水車小屋：** 這裡的水車是馬籠宿的標誌性景觀之一。

## 交通資訊

從 [JR 中津川站] 搭乘北惠那交通巴士約 25 分鐘即可抵達馬籠。詳細班次請參考 **[北惠那交通官方網站](https://www.kitaena.co.jp/)**。可搭配 **[昇龍道巴士](/日本/愛知/交通/昇龍道巴士)** 規劃行程。
