# 雪之大谷

鄰近立山黑部阿爾卑斯山脈路線中的室堂站，可從室堂站出站步行前往雪之大谷。雪之大谷僅在立山黑部阿爾卑斯山脈路線開山期間（通常為四月至十一月期間）開放參觀。關於立山黑部阿爾卑斯山脈路線開山詳細情況可參考[官方網站](https://www.alpen-route.com/tw/)

- 地址 〒930-1406 富山県中新川郡立山町芦峅寺ブナ坂 11
- 電話號碼 +81764811500
- 交通手段 鄰近黑部阿爾卑斯山脈路線室堂站，可從室堂站步行前往。
