# 雨晴海岸

位於 JR 冰見線的雨晴站附近的海岸。天氣晴朗時可觀賞遠處的北阿爾卑斯山脈，是富山縣的著名景點。

- 地址 〒933-0133 富山県高岡市太田 17
- 交通手段 搭乘 JR 冰見線抵達雨晴站，出站後步行即可抵達雨晴海岸。
