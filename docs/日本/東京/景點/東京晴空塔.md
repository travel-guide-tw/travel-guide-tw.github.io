# 東京晴空塔

東京晴空塔是世界第一高塔，高度達 634 公尺。塔內設有兩座觀景台，分別位於 350 與 450 公尺高處，可 360 度俯瞰東京全景甚至遠眺富士山。塔下的「晴空街道」包含商場、水族館與多樣化的餐廳。

- 地址：〒 131-0045 東京都墨田區押上 1 丁目 1-2
