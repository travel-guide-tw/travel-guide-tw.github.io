# 澀谷十字路口

澀谷十字路口被譽為「全世界最繁忙的十字路口」，在繁忙時段，每次綠燈亮起都有數千人同時橫越。這裡象徵著東京的活力與繁華，周邊環繞著大型百貨、LED 螢幕與八公犬像，是旅遊東京必訪的現代奇觀。

- 地址：〒 150-0043 東京都澀谷區道玄坂 2 丁目 2-1
