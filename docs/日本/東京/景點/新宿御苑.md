# 新宿御苑

新宿御苑原為皇室庭園，現已成為東京最受歡迎的都市公園之一。公園內融合了日式庭園、英式風景庭園及法式修整庭園等多種風格，是四季皆宜的散步勝地，特別是春季的櫻花與秋季的紅葉極為壯觀。

- 地址：〒 160-0014 東京都新宿區內藤町 11
