# 東京迪士尼樂園 & 東京迪士尼海洋

東京迪士尼度假區包含兩座風格迥異的主題樂園。東京迪士尼樂園以經典夢幻故事為背景，而東京迪士尼海洋則以海洋傳說為主題，擁有許多世界獨有的遊樂設施。這裡是大人小孩都能盡情玩樂的奇幻世界。

- 地址：〒 279-0031 千葉縣浦安市舞濱 1-1
