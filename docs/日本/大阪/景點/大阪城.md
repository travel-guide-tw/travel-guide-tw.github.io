# 大阪城公園

這裡是大阪最著名的櫻花景點之一，擁有超過 4000 棵櫻花樹，包括約 600 棵不同品種的櫻花樹。在櫻花季節，大阪城公園的夜間櫻花照明秀也很受歡迎。

- 地址：〒 540-0002 大阪市中央区大阪城 1 − 1
