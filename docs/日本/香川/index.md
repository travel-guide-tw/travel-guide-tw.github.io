# 香川

## 烏龍麵之鄉

香川縣位於四國地區，是日本面積最小的縣，但擁有豐富的自然景觀與深厚的文化底蘊，並以美味的烏龍麵享譽全國。

## 主要區域與景點

- **[高松](./景點/高松城跡)：** 香川縣的中心，擁有美麗的栗林公園與瀨戶大橋門戶。
- **[琴平](./景點/金刀比羅宮)：** 擁有著名的「金刀比羅宮」。
- **[小豆島](./景點/小豆島)：** 瀨戶內海上的美麗島嶼，以橄欖與夕陽聞名。
- **[藝術之島](./景點/直島)：** 直島等藝術景點。

## 鄰近區域推薦

香川是進入四國的大門，與周邊縣市連結緊密：
- **[岡山縣](../岡山/)：** 經由瀨戶大橋，搭乘電車僅需約一小時。
- **[愛媛縣](../愛媛/)：** 連結四國兩大門戶，體驗道後溫泉。
- **[德島縣](../德島/)：** 前往鳴門欣賞壯觀的漩渦（內容編寫中）。

## 溫馨提示

- 香川縣是烏龍麵的天堂，隨處可見美味的烏龍麵店。
- 瀨戶內國際藝術祭期間，前往各小島的交通需提前確認。
