# 昌古納拉揚神廟 (Changu Narayan Temple)

Changu Narayan 古老的印度教寺廟位於也被稱為 Changu 或 Dolagiri 的高山頂上。寺廟周圍環繞著佔木森林和一個名為 Changu 的小村莊。這座寺廟位於尼泊爾巴克塔普爾區昌古納拉揚市。
