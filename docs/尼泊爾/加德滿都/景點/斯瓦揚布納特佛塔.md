# 斯瓦揚布納特佛塔 (Swayambhunath)

斯瓦揚布納特（尼瓦爾語：स्वयंभू；尼泊爾語天城文स्वयम्भूनाथ，英語：Swayambhunath）位於尼泊爾加德滿都以西的加德滿都谷地，是一座佛教寺廟，也是尼泊爾國家象徵。又叫斯瓦揚布寺、四眼天神廟。

可以從西側的小山頂觀看山谷。Swyambhu 也被稱為 Samhengu 和猴廟，因為有無數的猴子住在這裡。
