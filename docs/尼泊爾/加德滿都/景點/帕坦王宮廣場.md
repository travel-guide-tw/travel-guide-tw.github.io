# 帕坦王宮廣場 (Patan Durbar Square)

帕坦王宮廣場 - 位於拉利特布爾市中心。它展示了在馬拉國王統治期間達到頂峰的紐瓦里建築，他們是偉大的建造者和藝術贊助人。
