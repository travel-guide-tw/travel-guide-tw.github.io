# 帕舒帕蒂納特廟 (Pashupatinath Temple)

帕舒帕蒂納特廟（尼泊爾語：पशुपतिनाथ मन्दिर），位於尼泊爾加德滿都東部巴格馬蒂河畔，是一座印度教寺廟，也是印度次大陸四大供奉濕婆的寺廟之一，創建時間可追溯至公元 400 年。
