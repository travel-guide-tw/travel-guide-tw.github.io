# 加德滿都皇宮廣場 (Kathmandu Durbar Square)

加德滿都皇宮廣場位於前加德滿都王國舊王宮前，是尼泊爾加德滿都谷地的三個杜巴廣場之一，均被聯合國教科文組織列為世界遺產。
