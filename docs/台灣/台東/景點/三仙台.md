# 三仙台

![三仙台](https://upload.wikimedia.org/wikipedia/commons/1/1f/Sanxiantai%2C_Taitung_2018.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

三仙台位於台東縣成功鎮東北方，是台東最具代表性的天然景觀。此地由突出的海岬與離岸小島組成，因島上有三座巨岩，民間傳說八仙中的呂洞賓、李鐵拐、何仙姑曾登臨此島而得名。整座島嶼被指定為自然保護區，擁有豐富的濱海植物與海蝕地貌。

## 景點特色

- **八拱跨海大橋**：銜接本島與珊瑚礁離岸島的波浪狀拱橋，外型如一條巨龍橫臥大海，是三仙台最鮮明的地標。
- **海蝕奇觀**：島上處處可見海蝕柱、海蝕溝及壺穴，展現大自然鬼斧神工的雕琢美感。
- **第一道曙光**：三仙台是全台灣甚至是全世界知名的迎曙光勝地，每年元旦都吸引萬人在此等待新年第一抹光暈。
- **豐富生態**：島上的濱海植物如林投、水芫花等生長茂盛，也是重要的鳥類棲息地。

## 門票資訊

- **參觀門票**：免費入場。
- **停車費用**：大型車 200 元、小型車 60 元、機車 20 元。

## 交通手段

- **公車**：自台東火車站或台東轉運站搭乘「台灣好行」東部海岸線或鼎東客運往成功、長濱方向之班車，於「三仙台」站下車。
- **自行開車**：沿台 11 線行駛至約 111 公里處即可抵達。
