# 台中捷運

台中捷運，簡稱中捷，是服務台中都會區的大眾捷運系統，為台灣中部的第一個捷運系統。

## 綠線

目前已通車營運的路線為綠線（烏日文心北屯線），全長約 16.71 公里，共 18 座車站。路線行經北屯、北區、西屯、南屯、南區、烏日等區域，並可與台鐵、高鐵轉乘。

## 未來發展

台中捷運有多條路線正在規劃與興建中，其中藍線已獲行政院核定，預計將串連市區與台中港。
