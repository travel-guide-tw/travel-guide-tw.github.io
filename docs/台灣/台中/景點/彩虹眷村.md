# 彩虹眷村

![彩虹眷村](https://upload.wikimedia.org/wikipedia/commons/c/c1/%E5%BD%A9%E8%99%B9%E7%9C%B7%E6%9D%91_%2810927199416%29.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

彩虹眷村位於台中市南屯區春安里，是一處色彩斑斕的眷村角落。由人稱「彩虹爺爺」的榮民黃永阜先生，在面臨拆遷之際拿起畫筆，在眷村的牆面、地面上繪滿了繽紛的動植物與人物畫作。這位素人藝術家的作品充滿童趣與活力，不僅成功將這塊舊社區保留下來，更使其成為國際級的熱門打卡景點。

## 景點特色

- **純真畫作**：牆面上繪有各類可愛的人物標語、傳統民俗意象與動物插畫，反映了爺爺樸實與善良的心境。
- **色彩饗宴**：大膽的艷紅、翠綠與湛藍色調相互交織，形成全世界獨一無二的露天彩繪美術館。
- **國際美譽**：曾獲選英國旅遊網站、日本網友最想去的拍照點，並獲選《孤獨星球》評選全球最值得探訪的景點之一。

## 門票資訊

- **參觀門票**：免費入場（特定時期採預約制開放，請依官網公告為準）。

## 交通手段

- **自行開車**：由國道 1 號下「南屯交流道」，往龍井方向行駛，循嶺東南路指標即可抵達。
- **公車**：搭乘台中市公車（如 27、56、127 路）至「彩虹眷村」站下車。
