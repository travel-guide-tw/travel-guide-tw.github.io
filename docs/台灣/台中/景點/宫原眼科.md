# 宮原眼科

宮原眼科位於台中市中區，原為日治時期的眼科醫院，現已華麗轉身為結合伴手禮、冰淇淋與下午茶的特色商店。其復古的建築風格和精緻的商品包裝，吸引了眾多遊客前來拍照打卡。

## 特色

- **冰淇淋：** 提供多種獨特口味的冰淇淋，搭配豐富配料，是必嚐甜點。
- **伴手禮：** 販售各式糕點、巧克力、茶葉等，包裝精美，送禮自用兩相宜。
- **建築風格：** 保留日治時期紅磚瓦建築特色，內部裝潢充滿懷舊氛圍。

## 交通資訊

位於台中火車站附近，步行即可抵達，交通便利。

## 溫馨提示

- **排隊人潮：** 假日人潮眾多，建議提早前往或避開尖峰時段。
- **商品價格：** 商品設計精美，價格相對較高。
