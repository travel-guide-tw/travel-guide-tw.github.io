# 逢甲夜市

逢甲夜市是台中市西屯區的一個大型觀光夜市，以創新和多樣化的台灣小吃聞名，是台灣最具指標性的夜市之一。夜市周邊商店林立，提供服飾、美妝、生活用品等多元購物選擇。

## 必吃美食

- **大腸包小腸：** 烤香腸搭配糯米腸，口感豐富。
- **懶人蝦：** 獨特調味，免剝殼即可享用。
- **明倫蛋餅：** 現桿現做的古早味蛋餅。
- **官芝霖大腸包小腸：** 逢甲夜市的排隊名店。

## 交通資訊

可搭乘公車至逢甲大學站，或自行開車前往，夜市周邊有收費停車場。

## 溫馨提示

- **人潮眾多：** 假日或晚間人潮擁擠，請注意隨身物品。
- **營業時間：** 大部分攤位營業至深夜，部分商店可能較早打烊。
