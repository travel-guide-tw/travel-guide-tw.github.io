# 高美濕地

![高美濕地](https://upload.wikimedia.org/wikipedia/commons/1/1c/Gaomei_Wetland%2C_Taichung%2C_Taiwan_%28555594035%29_3.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

高美濕地位於台中市清水區大甲溪出海口南側，是一處面積廣達 700 多公頃的濕地生態保護區。這裡擁有台灣稀有的雲林莞草，並以絕美的水平面夕陽倒影，被譽為「世界七大天空之鏡」之一。

## 景點特色

- **天空之鏡**：退潮後的薄水面能完美反射天空雲彩，夕陽時分展現如幻如夢的倒影景致。
- **生態木棧道**：長約 691 公尺的木棧道能讓遊客深入濕地核心而不破壞灘地，近距離觀察招潮蟹、彈塗魚及各式冬候鳥。
- **高美風車**：成排的巨型風力發電機是當地顯著的地標，在落日餘暉下構成壯觀的工業與自然美景。
- **豐富生態**：這裡是重要的鳥類棲息地，每年秋冬季吸引大量稀有候鳥在此棲息。

## 門票資訊

- **全區參觀**：免費。
- **木棧道開放**：每日視潮汐時間開放，建議出發前查閱「高美濕地旅遊網」潮汐表。

## 交通手段

- **高鐵/火車**：搭乘至台中火車站或高鐵台中站，轉乘公車（如 309 路）直達高美濕地。
- **火車 (海線)**：搭乘至「清水火車站」，轉乘 178、179 路公車。
- **自行開車**：國道 4 號清水端下，往省道台 17 線行駛即可抵達。
