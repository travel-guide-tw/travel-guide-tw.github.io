# 台中

台中位於台灣中部，是充滿藝術氣息與創意美食的城市。這裡氣候宜人，擁有許多特色的建築、公園和世界知名的夜市。

## 宜居之城的創意生活

台中市位於台灣中部，以其宜人的氣候、悠閒的生活步調和充滿現代感的城市設計著稱。這裡是台灣的文化與休閒中心，擁有前衛的表演藝術場館、世界級的自然步道以及讓許多美食愛好者神往的餐飲文化。

## 必訪景點

- **[逢甲夜市](./景點/逢甲夜市.md)：** 台灣最具影響力的創意美食發源地，也是規模最大的夜市之一。
- **[高美濕地](./景點/高美濕地.md)：** 擁有壯觀的風車背景與豐富的生態，是全台觀看夕陽的最美地點之一。
- **[台中國家歌劇院](./景點/台中國家歌劇院.md)：** 由普立茲克獎得主伊東豊雄設計，曲面建築工法令人嘆為觀止。
- **[彩虹眷村](./景點/彩虹眷村.md)：** 充滿繽紛色彩的彩繪眷村，由彩虹爺爺一手打造。
- **[宮原眼科](./景點/宫原眼科.md)：** 由老建築改裝而成的精緻伴手禮與冰淇淋店，內裝充滿魔法學院般的奇幻感。
- **[武陵農場與大雪山](./景點/武陵農場.md)：** 遠離塵囂的高山祕境，是賞櫻、賞秋楓與觀察高山生態的最佳去處。

## 交通資訊

台中是台灣中部的交通樞紐。旅客可搭乘高鐵至台中站，再轉乘捷運綠線或台鐵進入市中心。台中市公車系統提供「市民限定」的優惠，對遊客而言，公車網絡亦非常密布。

## 溫馨提示

- **美食推薦：**
  - **太陽餅：** 台中著名的傳統糕點，皮薄酥脆、內餡軟 Q。
  - **珍珠奶茶：** 台中被公認為珍奶的發源地，務必品嚐道地的始祖品牌。
  - **燒肉：** 台中的頂級燒肉餐廳競爭激烈，提供高品質的用餐體驗。
- **熱門活動：**
  - **后豐鐵馬道踏青：** 穿過舊火車隧道與壯觀的花樑鋼橋，享受騎行樂趣。
  - **審計新村慢生活：** 走訪青創市集，發掘在地文創好物。
- 台中氣候相對穩定乾燥，但冬季早晚溫差較大，建議採洋蔥式穿法。
