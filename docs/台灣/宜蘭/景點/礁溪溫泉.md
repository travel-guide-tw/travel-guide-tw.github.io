# 礁溪溫泉

礁溪溫泉位於宜蘭縣礁溪鄉，是台灣著名的溫泉區，以其無色無味、富含鈉、鎂、鈣、鉀離子及碳酸氫根的碳酸氫鈉泉質而聞名。泡湯後皮膚光滑，因此有「美人湯」之稱。

## 泡湯體驗

- **溫泉飯店：** 礁溪溫泉區飯店林立，提供多樣的溫泉住宿選擇。
- **溫泉公園：** 設有免費泡腳池，讓遊客輕鬆享受溫泉樂趣。
- **湯圍溝溫泉公園：** 結合日式庭園造景，設有免費泡腳池和付費湯屋。

## 交通資訊

搭乘火車至礁溪車站，或從台北搭乘國道客運，皆可直達礁溪溫泉區。

## 溫馨提示

- **溫泉禮儀：** 泡湯前請先淋浴，勿在溫泉池中搓洗身體。
- **周邊美食：** 礁溪有許多特色小吃，如溫泉拉麵、宜蘭蔥油餅等。
