# 宜蘭

宜蘭擁有得天獨厚的蘭陽平原美景，是台北人休假時的首選。這裡以溫泉、冷泉、豐富的農業觀光和蘭陽溪的美景著稱。

## 熱門景點

- **[礁溪溫泉](./景點/礁溪溫泉.md)**
- **[太平山國家森林遊樂區](./景點/太平山.md)**
- **[蘭陽博物館](./景點/蘭陽博物館.md)：** 單面山建築融入地景，展示宜蘭的自然與人文歷史。
- **[國立傳統藝術中心](./景點/國立傳統藝術中心.md)：** 保存、傳習、推廣台灣傳統藝術文化的重要基地。
- **[羅東夜市](./景點/羅東夜市.md)：** 宜蘭最受歡迎的夜市之一，以豐富多樣的傳統風味小吃聞名。

## 交通資訊

從台北可搭乘葛瑪蘭客運或首都客運，或搭乘台鐵區間快車，穿過雪山隧道或沿著東北角海岸到達。
