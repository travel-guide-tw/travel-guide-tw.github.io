# 高雄

高雄是台灣南部的門戶，也是一個海港都市。這裡擁有寬闊的大道、充滿活力的駁二藝術特區和美麗的西子灣夕陽。

## 港都之心的藝術與活力

高雄市是台灣南部的門戶，也是一個充滿陽光與海港氣息的都會。這裡擁有寬闊的高雄港風景、獨具特色的捷運藝術站，以及由舊倉庫群轉型而成的文創特區，展現了從重工業城市轉型為美學都市的驚人生命力。

## 必訪景點

- **[蓮池潭](./景點/蓮池潭.md)：** 以龍虎塔、春秋閣等湖畔寺廟聞名的風景區。
- **[駁二藝術特區](./景點/駁二藝術特區.md)：** 位於碼頭旁的舊倉庫群，現在是高雄最重要的當代藝術與設計展示空間。
- **[旗津島](./景點/旗津島.md)：** 從渡輪站搭船前往，享受海鮮大街風情、旗後砲台古蹟與黑沙灘。
- **[西子灣與打狗英國領事館](./景點/西子灣.md)：** 海天一色的夕陽絕景，以及俯瞰高雄港的歷史建築。
- **[瑞豐夜市](./景點/瑞豐夜市.md)：** 在地人首選的大型夜市，規模龐大且創意小吃林立。
- **[佛光山佛陀紀念館](./景點/佛光山.md)：** 宏偉的佛教聖地，以壯觀的佛像建築與和平的氣氛吸引眾多國內外遊客。

## 交通資訊

高雄擁有完善的高雄捷運 (KMRT) 紅線與橘線，以及全台第一條環狀輕軌 (LRT)，可輕鬆到達主要景點景點。從台北搭乘高鐵只需約 1.5 - 2 小時即可到達左營站。高雄港區也可利用渡輪往返旗津。

## 溫馨提示

- **美食推薦：**
  - **海鮮料理：** 旗津與港區附近可品嚐到當日捕撈的平價鮮美海鮮。
  - **鴨肉飯：** 高雄有多家歷史悠久的鴨肉飯老舖，是不可錯過的在地味。
  - **木瓜牛奶：** 高雄夜市的招牌飲品之一。
- **熱門活動：**
  - **環港單車行：** 租借 CityBike 沿著港岸自行車道騎乘，享受徐徐海風。
  - **愛河遊船：** 搭乘太陽能「愛之船」欣賞夜晚愛河兩岸的浪漫燈火。
- 高雄氣候炎熱且陽光強烈，請隨時注意防曬與水分補充，即使是冬季也相當溫暖。
