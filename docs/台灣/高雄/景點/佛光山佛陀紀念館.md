# 佛光山佛陀紀念館

![佛光山佛陀紀念館](https://upload.wikimedia.org/wikipedia/commons/2/23/Fo_Guang_Shan_Buddha_Memorial_Center02.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

佛光山佛陀紀念館位於高雄市大樹區，座西朝東，占地 100 餘公頃。其興建是為供奉佛陀真身舍利，是一座融和古今、中印文化，建基於傳統佛教精神並結合現代藝術、教育、休閒功能的博物館建築。

## 景點特色

- **佛光大佛**：全館的地標，高約 108 公尺，是世界最高的銅鑄坐佛，慈悲莊嚴。
- **成佛大道與八塔**：大道兩側矗立著八座代表佛法核心的中國式寶塔，莊重典雅。
- **本館**：內部設有佛牙舍利殿，並擁有多個常設展廳與特展空間，展示佛教珍貴文物。
- **禮敬大廳**：入館首站，提供豐富的蔬食餐飲與文創商店服務。

## 門票資訊

- **參觀門票**：免費入場。

## 交通手段

- **高鐵轉乘**：自高鐵左營站搭乘義大客運 8501 路「哈佛快線」直達佛陀紀念館。
- **火車轉乘**：搭乘至「九曲堂車站」，轉乘高雄客運 8501 路。
