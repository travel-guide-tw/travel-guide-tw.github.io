# 駁二藝術特區

![駁二藝術特區](https://upload.wikimedia.org/wikipedia/commons/3/3f/The_Pier-2_Art_Center_2020070902.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

駁二藝術特區位於高雄港第三船渠內，原為存放魚粉與糖的舊倉庫群。2001 年在一群熱心藝文人士推動下，將廢棄倉庫活化轉型，如今已成為高雄最具代表性的前衛、實驗、與創意的文化藝術園區。

## 景點特色

- **倉庫活化**：由大勇、大義、蓬萊三區倉庫群組成，將歷史空間轉化為展演場地與創意基地。
- **戶外裝置藝術**：園區內隨處可見大型公仔、彩繪與雕塑作品（如「大黃蜂」），是熱門的拍攝打卡點。
- **文創市集與展覽**：定期舉辦各種主題展覽、動漫節、大港開唱及手作市集。
- **鐵道公園**：蓬萊區保留了舊哈瑪星鐵道軌跡，提供廣大的草原供民眾休閒。

## 門票資訊

- **園區進入**：免費
- **特定展覽**：需另購門票。
- **駁二護照手環**：可一日或二日通票參觀指定場館，詳情請洽官網。

## 交通手段

- **捷運**：搭乘高雄捷運橘線至「鹽埕埔站」（O2 站）1 號出口，步行約 5 分鐘即可抵達大勇區。
- **輕軌**：搭乘輕軌至「駁二蓬萊站」（C13）或「駁二大義站」（C12）下車即達。
