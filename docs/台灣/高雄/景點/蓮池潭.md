# 蓮池潭

![龍虎塔](https://upload.wikimedia.org/wikipedia/commons/9/90/Dragon_and_Tiger_Pagodas_02.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

蓮池潭位於高雄市左營區，是一個面積廣大的人工湖泊。湖畔環繞著密集的寺廟與寶塔，其中最著名的莫過於「龍虎塔」，是高雄最具代表性的國際級觀光景點。

## 景點特色

- **龍虎塔**：蓮池潭的地標。遊客需遵從「從龍口進、由虎口出」的傳統，寓意為「消災增長福慧」。
- **春秋閣**：為紀念武聖關公而建，由兩座中國宮殿式樓閣組成，中間有九曲橋相連。
- **玄天上帝神像**：擁有世界最高的水上神像之一，氣勢磅礡。
- **環潭步道**：湖畔設有完善的環潭自行車道與散步道，並提供水上活動如滑水體驗。

## 門票資訊

- **全區開放**：免費參觀。

## 交通手段

- **捷運/高鐵**：搭乘捷運紅線或高鐵至「左營站」，步行約 15-20 分鐘或轉乘公車、使用共享單車前往。
- **台鐵**：搭乘至「左營（舊城）站」，出站後即可抵達湖畔。
