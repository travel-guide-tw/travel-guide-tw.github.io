# 台北燈節

![台北燈節](https://upload.wikimedia.org/wikipedia/commons/f/fb/2019_Taipei_Lantern_Festival_main_lantern_Mickey_Mouse.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

台北燈節是台北市政府每年於元宵節期間舉辦的大型慶典活動。自 1997 年起，這項活動已成為台北市民及國內外遊客引頸期盼的年度盛事。燈節不僅展示各種高科技與藝術美感兼具的各式燈組，更結合表演藝術與在地商圈，打造出一場華麗的視覺饗宴。

## 景點特色

- **主題主燈**：每年邀請知名藝術家設計具備當年生肖特色的主燈，結合音樂與光影投射，呈現震撼的感官體驗。
- **多元展區**：燈會通常涵蓋台北市的高樓大廈、舊城區或河濱公園，將城市空間轉化為巨大的戶外藝術廳。
- **踩街遊行**：大型的嘉年華遊行，包含花車表演與藝文團體演出，帶動整座城市的歡愉氣氛。
- **在地文化結合**：融入地方民俗特色，如西門町的多樣文化或大龍峒的傳統燈飾。

## 門票資訊

- **全區參觀**：免費。

## 交通手段

- **捷運**：活動地點通常選在捷運站周邊，如「西門站」、「北門站」、「市政府站」或「劍潭站」。詳細站點請依當年度公告。
- **公車**：多路公車可直達鄰近展區的接駁站點。
