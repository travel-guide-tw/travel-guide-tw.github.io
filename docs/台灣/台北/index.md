# 台北

台北是台灣的首都，也是一個充滿活力、融合傳統與現代的國際大都市。這裡有璀璨的摩天大樓、古老的寺廟、熱鬧的夜市和世界級的博物館。

## 融合現代與傳統的活力之都

台北市是台灣的政治、經濟與文化中心，也是一個充滿活力、傳統與現代交織的國際大都市。無論是璀璨的摩天大樓、歷史悠久的古老寺廟，還是香氣四溢的道地夜市，台北都能滿足每位旅客的期待。

## 必訪景點

- **[台北 101](./景點/台北101)：** 曾為世界最高樓，是台北的地標，擁有絕佳的觀景台。
- **[國立故宮博物院](./景點/國立故宮博物院)：** 收藏了數量豐富的中華藝術珍寶，是文化愛好者的朝聖地。
- **[中正紀念堂](./景點/中正紀念堂)：** 宏偉的國家紀念建築，以其廣闊的自由廣場和衛兵交接儀式而聞名。
- **[龍山寺](./景點/龍山寺)：** 台北最古老且著名的寺廟，建築精美且香火鼎盛。
- **[大稻埕與迪化街](./景點/大稻埕)：** 保留了大量古老建築，是體驗台北舊城區風情的好地方。
- **[陽明山國家公園](./景點/陽明山)：** 台北的後花園，提供溫泉、健行步道與壯麗的火山口景觀。

## 交通資訊

台北擁有全國最發達的台北捷運 (MRT) 路網，搭配密集的公車系統，交通極為便利。旅客可使用悠遊卡 (EasyCard) 輕鬆搭乘各類大眾運輸工具。從桃園國際機場可搭乘機場捷運直達台北車站。

## 溫馨提示

- **美食推薦：**
  - **士林夜市與饒河街夜市：** 必嚐胡椒餅、大餅包小餅、藥燉排骨等在地小吃。
  - **鼎泰豐：** 聞名全球的小籠包。
  - **牛肉麵：** 台北有許多獲獎的牛肉麵老店，是必吃的正餐選擇。
- **熱門活動：**
  - **象山健行：** 這裡有拍攝台北 101 與市景的最佳角度。
  - **北投泡湯：** 體驗歷史悠久的青磺與白磺泉。
- 台北夏季氣候炎熱且多雷雨，建議隨身攜帶雨具並注意防曬。
