# 台北捷運

台北捷運，又稱台北大眾捷運系統，是服務台北市與新北市的大眾捷運系統，為台灣第一個啟用、規模最大的捷運系統。

## 路線簡介

台北捷運目前有 5 條主要路線及 2 條支線，分別為：

- **文湖線（棕線）**
- **淡水信義線（紅線）**
- **松山新店線（綠線）**
- **中和新蘆線（橘線）**
- **板南線（藍線）**

此外，新北捷運環狀線（黃線）也與台北捷運系統整合。

## 特色與服務

台北捷運以其「安全、可靠、便捷、舒適」的服務品質著稱。全系統禁止飲食、吸菸，以維持車站及車廂的整潔。車站及列車內提供中、英、台、客四種語言廣播。
