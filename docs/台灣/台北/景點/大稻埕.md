# 大稻埕

大稻埕位於台北市大同區淡水河畔，曾是清末至日治時期重要的商業貿易中心。如今，這裡保留了許多歷史建築、傳統店家，並注入了文創活力，是體驗台北舊城風情的最佳地點。

## 必訪景點

- **迪化街：** 台北年貨大街，販售南北貨、中藥材、布料等，充滿懷舊氛圍。
- **霞海城隍廟：** 香火鼎盛的廟宇，以月老聞名，吸引許多單身男女前來祈求姻緣。
- **大稻埕碼頭：** 夕陽西下時分景色優美，可搭乘藍色公路遊船。

## 交通資訊

可搭乘捷運至大橋頭站或北門站，步行即可抵達。

## 溫馨提示

- **傳統店家：** 許多老店仍保有傳統製程，值得細細品味。
- **文創小店：** 穿梭巷弄間，可發現許多特色文創商品。
