# 和平島公園

基隆市的地質公園，也是基隆市的地標之一，公園內有著許多奇特的地質景觀，也有著許多的海岸生態，是一個很適合親子同遊的地方。

- 地址 基隆市中正區平一路 360 號
- 開放時間 5 月 -10 月｜08:00-19:00、11 月 -4 月｜08:00-18:00
