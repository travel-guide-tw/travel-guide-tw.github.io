# 司馬庫斯

司馬庫斯部落，被譽為「上帝的部落」，位於新竹縣尖石鄉的深山中，是一個泰雅族部落。這裡以保存完整的巨木群和原始森林生態聞名，是台灣最偏遠但同時也最受歡迎的觀光部落之一。

## 必訪景點

- **巨木群步道：** 徒步穿越原始森林，欣賞多棵千年神木。
- **部落文化體驗：** 了解泰雅族傳統文化、建築和生活方式。

## 交通資訊

司馬庫斯路途遙遠且部分路段狹窄，建議搭乘當地接駁車或具備高山駕駛經驗的司機前往。實施總量管制，需提前申請入山證。

## 溫馨提示

- **生態保育：** 請尊重當地生態環境，不亂丟垃圾。
- **住宿：** 部落內有民宿可供住宿，需提早預訂。
