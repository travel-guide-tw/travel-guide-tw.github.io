# 新竹都城隍廟

![新竹都城隍廟](https://upload.wikimedia.org/wikipedia/commons/4/49/%E6%96%B0%E7%AB%B9%E9%83%BD%E5%9F%8E%E9%9A%8D%E5%BB%9F.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

新竹都城隍廟位於新竹市北區，創建於清乾隆 13 年（西元 1748 年），是全台灣歷史最悠久、且官位最高的城隍廟（晉封為都城隍）。這座廟宇不僅是新竹市重要的信仰中心，更因其獨特的「廟中有市」格局聞名。廟前廣場匯集了數十家歷史悠久的小吃攤位，是品嚐道地新竹風味與體驗民俗活動的首選之地。

## 景點特色

- **官位最高的城隍**：清領時期獲得晉封，是全台唯一的「省級」城隍廟，地位極其尊崇。
- **建築雕刻藝術**：主體建築採三疊式屋頂，內部雕刻精巧、古意盎然，被列為市定古蹟。
- **廟埕小吃文化**：廟前著名的美食區有著名的「新竹三寶」：米粉、摃丸與肉圓，還有水蒸蛋糕、潤餅等在地傳統美味。
- **年度民俗盛事**：每年農曆七月的「城隍祭」包含了開虎門、查夜與放水燈等盛大儀式，吸引大批信徒參與。

## 門票資訊

- **參觀參拜**：免費。

## 交通手段

- **火車**：搭乘台鐵至「新竹火車站」，出站後由站前商圈步行約 10-15 分鐘即可抵達。
- **市區公車**：搭乘多路公車（如 50、51 路）於「城隍廟」站下車。
