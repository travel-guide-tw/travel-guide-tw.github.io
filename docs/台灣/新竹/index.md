# 新竹

新竹被譽為「台灣矽谷」，是高科技產業的重鎮。除了現代化的科學園區，新竹也有悠久的歷史、古老的廟宇和獨特的客家文化。

## 科技與風城之域

新竹被譽為「台灣矽谷」，是全球高科技產業的重鎮。但在冷冽的高科技外殼下，新竹保有悠久的歷史城廓、豐富的客家文化以及獨特的自然山林，是一個充滿驚喜的學習與休閒之城。

## 必訪景點

- **[新竹都城隍廟](./景點/新竹都城隍廟)：** 台灣位階最高的城隍廟，周邊環繞著各式各樣的百年在地小吃。
- **[小叮噹科學主題樂園](./景點/小叮噹科學主題樂園)：** 寓教於樂的科學主題樂園，內有全台最大室內滑雪場。
- **[十七公里海岸線](./景點/十七公里海岸線)：** 絕美的單車道，途經南寮漁港、香山濕地，是觀賞落日的絕佳位置。
- **[內灣老街](./景點/內灣老街)：** 體驗濃濃的客家風情，品嚐野薑花粽，還能搭乘懷舊的內灣支線小火車。
- **[司馬庫斯](./景點/司馬庫斯)：** 隱身尖石鄉深山的「上帝之部落」，以巨大的紅檜神木群著稱。
- **[六福村主題遊樂園](./景點/六福村主題遊樂園)：** 結合了野生動物園與多元主題遊樂設施，是全台最受歡迎的樂園之一。

## 交通資訊

新竹市可透過台鐵 (TRA) 或國道客運輕鬆到達。台灣高鐵新竹站位於竹北市，可轉乘六家線火車進入市中心。市內交通可利用便捷的公車系統或租借 YouBike。

## 溫馨提示

- **美食推薦：**
  - **新竹米粉與貢丸：** 受惠於獨特的「三分貢丸、七分風」，新竹的米粉與貢丸口感世界知名。
  - **水潤餅：** 僅存在於新竹的古早味點心，口感軟 Q。
  - **客家擂茶與麻糬：** 到內灣老街必嚐的傳統美味。
- **熱門活動：**
  - **南寮漁港放風箏：** 這裡風力強勁，是全台知名的放風箏聖地。
  - **新竹公園賞櫻：** 春季時這裡會變身成浪漫的粉紅世界。
- 新竹因地形關係在秋冬季節風勢強勁，前往旅遊時請準備防風外套。
