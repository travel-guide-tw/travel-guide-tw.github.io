# 墾丁國家公園

![墾丁大尖山](https://upload.wikimedia.org/wikipedia/commons/2/2c/Mt._Dajian%2C_Kenting.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

墾丁國家公園位於台灣最南端的恆春半島，是台灣首座成立的國家公園。這裡三面環海（巴士海峽、太平洋及台灣海峽），擁有獨特的熱帶氣候、發達的珊瑚礁地形及豐富的動植物生態，是台灣最具代表性的熱帶渡假勝地。

## 景點特色

- **鵝鑾鼻燈塔**：台灣最南端的燈塔，純白色的燈塔在蔚藍海景照射下顯得格外壯麗。
- **南灣與小灣**：金黃翠綠的沙灘是水上活動的天堂，適合浮潛、香蕉船及衝浪。
- **墾丁大街**：全台知名的觀光夜市，匯集了各式南洋風情料理、台式小吃與創意調酒。
- **森林與草原**：如龍磐公園的可一覽無遺的太平洋海景，以及墾丁國家森林遊樂區內珍貴的熱帶雨林與石灰岩遺址。

## 門票資訊

- **國家公園進入**：免費。
- **特定景點**：鵝鑾鼻公園（全票 60 元）、墾丁森林遊樂區（全票 150 元/假日）。

## 交通手段

- **高鐵轉乘**：自高鐵左營站搭乘「墾丁快線」公車至南保力站，轉乘屏東客運「墾丁街車」(橘線) 直達海生館。
- **自行開車**：經國道 3 號下枋山、車城方向即可抵達恆春半島。
