# 勝利星村創意生活園區

勝利星村創意生活園區位於屏東市，是台灣現存規模最大、保存最完整的日治時期陸軍官舍建築群。近年來經過活化再利用，轉型為結合文創、餐飲、藝文展演的多元空間。

## 特色店家

- **文創商品：** 許多獨立設計師進駐，販售特色手作商品。
- **特色餐飲：** 提供各國料理、咖啡廳、甜點店等，選擇豐富。
- **藝文空間：** 不定期舉辦展覽、講座、市集等活動。

## 交通資訊

屏東火車站搭乘計程車或公車即可抵達，步行亦可。

## 溫馨提示

- **歷史建築：** 漫步園區，感受歷史氛圍，欣賞日式建築之美。
- **假日活動：** 假日經常舉辦市集或表演活動，氣氛熱鬧。
