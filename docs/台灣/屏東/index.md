# 屏東

屏東位於台灣最南端，擁有熱帶氣候和迷人的海岸線。墾丁國家公園是台灣最知名的渡假勝地，提供陽光、沙灘和豐富的水上活動。

## 國境之南的熱帶風情

屏東縣位於台灣的最南端，擁有蔚藍的世界級海岸、豐富的排灣族與魯凱族原民文化，以及全台唯一的熱帶雨林氣候。無論是追求極限水上活動，還是尋找心靈寧靜的神祕山谷，屏東都能提供最純粹的高熱量驚喜。

## 必訪景點

- **[墾丁國家公園](./景點/墾丁國家公園)：** 台灣首座國家公園，擁有絕美沙灘、裙礁海岸與熱鬧的墾丁大街。
- **[鵝鑾鼻燈塔](./景點/鵝鑾鼻)：** 台灣最南端的燈塔，潔白塔身與壯闊海景相映，是著名的拍照聖地。
- **[國立海洋生物博物館](./景點/國立海洋生物博物館)：** 台灣規模最大的海洋生物博物館，有豐富的海洋生物展示和海底隧道。
- **[小琉球](./景點/小琉球)：** 台灣唯一的珊瑚礁島嶼，以其獨特的海洋生態、清澈的海水和豐富的潮間帶生物而聞名。
- **[勝利星村創意生活園區](./景點/勝利星村)：** 規模龐大的歷史建築群遺產，現在注入了無數設計力與創意美食。

## 交通資訊

前往屏東通常先搭乘高鐵至左營站，再轉乘「墾丁快線」客運或租車前往。屏東市區則可搭乘台鐵至屏東車站。往返小琉球需從東港或鹽埔搭乘船隻前往。

## 溫馨提示

- **美食推薦：**
  - **後壁湖海鮮：** 以平價且新鮮度驚人的生魚片聞名。
  - **萬巒豬腳：** 皮 Q 肉嫩並搭配獨門蒜蓉醬，是屏東最知名的特產。
  - **東港三寶：** 黑鮪魚、櫻花蝦、油魚子，海產愛好者的夢幻名單。
- **熱門活動：**
  - **水上活動：** 衝浪、潛水、獨木舟，盡情享受墾丁的海天一色。
  - **原民部落體驗：** 走訪霧台、三地門，體驗深厚的原民美學與工藝。
- 墾丁地區在連續假期與暑假期間人潮眾多，建議提前數月預訂住宿與交通。
