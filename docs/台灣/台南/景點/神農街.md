# 神農街

![神農街](https://upload.wikimedia.org/wikipedia/commons/d/d4/%E8%87%BA%E5%8D%97%E5%8C%97%E5%8B%A2%E8%A1%97.JPG)
_圖片來源：Wikimedia Commons_

## 簡介

神農街，舊名北勢街，是台南市五條港文化園區中保存最完善、也最具特色的一條清代老街。街道寬度僅約四公尺，兩側矗立著兩層高的古老木造建築，曾是台南最重要的商業進出口區。如今，這條街道已成為文創、藝術與老屋活化的代名詞。

## 景點特色

- **五條港門戶**：街道盡頭為「藥王廟」，是昔日貿易往來的重要口岸，老屋前門臨街、後門通河的獨特格局依稀可辨。
- **復古藝術街區**：許多藝術工作者與文創店家進駐，在保留古宅骨架的基礎上，加入充滿個性的招牌與裝飾。
- **夜間燈飾美景**：入夜後整條街的紅色燈籠點亮，映照著質樸的木門與春聯，散發出極其迷人的懷舊氣息。
- **特色文創小店**：包含手作皮件、古物店、特色咖啡館與深夜酒吧，每一家店都各有千秋。

## 門票資訊

- **全街開放**：免費參觀（個別店家依營業項目收費）。

## 交通手段

- **公車**：搭乘台南市區公車至「神農街」站或「民權路口」站下車，步行即可抵達。
- **自行前往**：位於西門路與海安路之間，建議將車輛停放在海安路地下停車場再行前往。
