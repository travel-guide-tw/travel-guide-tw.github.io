# 安平古堡

![安平古堡](https://upload.wikimedia.org/wikipedia/commons/a/a8/Anping_Fort.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

安平古堡，古稱熱蘭遮城，是台灣歷史上第一座城堡，由荷蘭人於 1624 年所建。它曾是荷蘭貿易中樞與台灣政治核心，也曾是鄭氏王朝三代的宅邸。歷經數百年的洗禮，如今名列國家一級古蹟，是見證台灣大航海時代的重要史蹟。

## 景點特色

- **熱蘭遮城城垣殘蹟**：保留了荷蘭時期的古城牆殘蹟，紅磚疊砌出的斑駁美感，訴說著四百年的歷史。
- **安平古堡瞭望台**：雖然瞭望台是日治時期改建而成，但登上塔頂後可俯瞰整座安平老街與海岸線。
- **安平稅務司公署**：位於城堡遺址旁的西式平房建築，現作為史料陳列館使用。
- **古砲陳列**：園區內展示多座清領時期的古砲，增添了城堡的軍事防禦氣息。

## 門票資訊

- **全票**：新台幣 70 元
- **半票**：新台幣 35 元
- **優惠對象**：台南市民憑身分證件免費。

## 交通手段

- **公車**：自台南火車站搭乘 2 路、88 路 (台灣好行大安平線) 或 99 路 (台灣好行安平台江線) 公車至「安平古堡」站。
- **自行開車**：可將車輛停放於安平區運河旁或老街周邊停車場，步行入園。
