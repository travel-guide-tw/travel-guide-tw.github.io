# 桃園

桃園是台灣的門戶，擁有台灣最重要的國際機場。除了作為交通樞紐，桃園也擁有豐富的自然景觀與文化特色。

## 台灣的門戶與山海之鄉

桃園市不僅是台灣最重要的國際交通樞紐，更是一個融合了多元族群文化與壯麗自然景觀的城市。從充滿歷史氣息的大溪老街到高山美景拉拉山，桃園展現了深厚的文化底蘊與動人的自然魅力。

## 必訪景點

- **[大溪老街](./景點/大溪老街)：** 以精美的巴洛克風格建築和道地的豆干美食聞名。
- **[慈湖陵寢](./景點/慈湖陵寢)：** 蔣公陵寢與紀念園區，有衛兵交接儀式與充滿江南風情的庭園。
- **[石門水庫](./景點/石門水庫)：** 北台灣重要的水利工程，也是賞楓、單車旅行與品嚐「活魚多吃」的勝地。
- **[Xpark 水族館](./景點/Xpark水族館)：** 位於青埔的都會型水族館，以日系美學呈現奇幻的海洋世界。
- **[拉拉山 (達觀山)](./景點/拉拉山)：** 擁有巨大的千年神木群，夏季更是品嚐甜美水蜜桃的最佳地點。
- **[小人國主題樂園](./景點/小人國主題樂園)：** 適合親子同遊，展示了精緻的微縮景觀與遊樂設施。

## 交通資訊

桃園國際機場位於本市中心，可透過機場捷運串連台北與桃園市區。台灣高鐵桃園站則提供快速往返中南部的選擇。市內旅行建議利用公車系統、租車或搭乘台鐵區間車。

## 溫馨提示

- **美食推薦：**
  - **大溪豆干：** 各式風味的豆干是桃園最具代表性的伴手禮。
  - **活魚料理：** 石門水庫附近的餐廳提供豐富的活魚多吃體驗。
  - **龍岡米干：** 體驗獨特的雲南穆斯林餐飲文化。
- **熱門活動：**
  - **石門水庫騎單車：** 享受環湖美景與清新空氣。
  - **觀音草漯沙丘：** 台灣版的撒哈拉沙漠，是熱門的拍照打卡點。
- 桃園的海岸線擁壯觀的藻礁景觀，建議配合潮汐時間前往觀賞。
