# 慈湖陵寢

![慈湖陵寢 外部景觀](https://upload.wikimedia.org/wikipedia/commons/8/81/Cihu_Mausoleum.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

慈湖陵寢位於桃園市大溪區，是中華民國先總統蔣中正先生的靈柩暫厝之地。這裡四周環山，兩側溪流交匯成湛藍如鏡的湖泊，據說因地形酷似蔣先生的故鄉浙江奉化溪口鎮，因而被選為紀念之地。園區結合了自然湖景、歷史紀念與莊嚴的軍事典儀，是北台灣知名的觀光勝地。

## 景點特色

- **三軍儀隊衛兵交接**：最受遊客矚目的定時儀隊交接儀式，動作精準、莊嚴肅穆，展現了高超的操槍技術與威武軍容。
- **前慈湖步道**：沿著碧綠的湖岸漫步，欣賞垂柳與黑天鵝在湖面嬉戲，感受江南庭園般的清幽氣息。
- **慈湖紀念雕塑公園**：園內陳列了來自全台各地捐贈、超過兩百座形式各異的蔣公遺像與雕塑，形成獨特的當代歷史地景。
- **後慈湖探索**（需預約）：曾為高度保密的軍事管制區，保存了完整的原始林相與戰時指揮所防空建築。

## 門票資訊

- **參觀門票**：免費入場（後慈湖探秘需事先預約導覽費用約 100 元）。

## 交通手段

- **大眾運輸**：由桃園客運中壢總站或桃園站撘乘 501 路「台灣好行大溪快線」(假日) 至終點「慈湖站」即可抵達。
- **自行開車**：行經國道 3 號下大溪交流道，沿台 4 線接台 7 線（北橫公路）往大溪、慈湖方向前進。
