# 大溪老街

![大溪老街 巴洛克建築](https://upload.wikimedia.org/wikipedia/commons/5/55/Daxi_Old_Street.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

大溪老街位於桃園市大溪區，曾是大漢溪航運與茶葉貿易的重要集散中心。至今老街兩旁的和平路、中山路與中央路上，仍保存著日治時期流行的大量精緻「巴洛克式建築」立面，結合了西式的繁複雕繪與中式的祥瑞圖案。這裡不僅展現了往昔商業鼎盛的繁華，更以其獨特的「木藝文化」與盛名的「豆干美食」聞名全台。

## 景點特色

- **巴洛克式立面建築**：一整排保存完好的牌樓立面，精雕細琢的希臘神像、西方紋章與中國魚躍龍門等浮雕交相輝映，是建築攝影的絕佳去處。
- **大溪木藝生態博物館**：將歷史建築群轉化為開放式博物館，推廣大溪歷史悠久的精緻木藝工法與職人精神。
- **豆干小吃與童玩**：街頭到處可見熱騰騰的現滷豆干攤位，還有月光餅、花生糖等道地特產，是老少咸宜的小吃勝地。
- **大溪橋美景**：連結老街與大漢溪對岸的仿巴洛克式觀光橋樑，夜晚點燈後如夢似幻。

## 門票資訊

- **老街參觀**：免費參觀。

## 交通手段

- **大眾運輸**：可於桃園火車站搭乘「5096 路公車」或於中壢火車站搭乘「5098 路公車」至「大溪總站」下車，步行至老街。
- **台灣好行**：搭乘「大溪快線」(假日) 直達大溪。
- **自行開車**：經由國道 3 號下大溪交流道，循指標往大溪市區前進，可將車輛停放在橋頭停車場。
