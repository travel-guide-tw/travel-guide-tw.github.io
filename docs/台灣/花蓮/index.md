# 花蓮

花蓮面臨太平洋，背倚中央山脈，擁有壯麗的山海奇景。太魯閣國家公園的峽谷景觀世界聞名，是自然愛好者的天堂。

## 熱門景點

- **[太魯閣國家公園](./景點/太魯閣國家公園)：** 以雄偉壯麗的峽谷和多樣的生態而聞名，是世界級的自然景觀。
- **[七星潭](./景點/七星潭)：** 優美的弧形海灣，海水潔淨湛藍，是觀星和賞日出的絕佳地點。
- **[東大門夜市](./景點/東大門夜市)：** 花蓮的知名觀光夜市，提供多元美食與娛樂。
- **[瑞穗牧場](./景點/瑞穗牧場)：** 綠意盎然的牧場，可品嚐香醇鮮乳製品，與動物近距離接觸。

## 交通資訊

前往花蓮建議搭乘台鐵 (如普悠瑪號或太魯閣號) 從台北出發，約需 2-3 小時。
