# 太魯閣國家公園

![太魯閣峽谷 絕景](https://upload.wikimedia.org/wikipedia/commons/f/f2/Taroko_National_Park.jpg)
_圖片來源：Wikimedia Commons_

## 簡介

太魯閣國家公園位於台灣東部，橫跨花蓮、南投及台中三縣市。其最富盛名的景觀當屬「大理岩峽谷」，由立霧溪千萬年來不斷向下切蝕而成的峽谷深邃壯闊。這裡匯集了豐富的泰雅族原住民文化、多樣性的生態體系與世界級的斷崖地貌（如清水斷崖），是台灣在國際上最著名的國家級自然觀光名片。

## 景點特色

- **燕子口與九曲洞**：這兩處步道能近距離觀察峽谷中最窄窄且雄偉的路段，看那大理石壁上的奇特壺穴景觀，以及立霧溪水激盪出的咆哮聲。
- **清水斷崖**：蘇花公路上高達數百公尺的懸崖陡坡直插入太平洋，海天一色的漸層藍調展現了驚心動魄的美。
- **天祥與白楊步道**：步道盡頭有著名的「水簾洞」，山泉水由隧道頂端如幕簾般灑落，是十分消暑的探險行程。
- **長春祠**：依山傍水的中式涼亭建築，紀念當年開闢中橫公路的築路英雄，泉水自山壁流出形成流瀑。

## 門票資訊

- **全區區入**：免費（部分登山步道與入山證需事先預約申請）。

## 交通手段

- **大眾運輸**：於花蓮火車站或新城（太魯閣）火車站搭乘「台灣好行」太魯閣線 (1133) 直達園內各大景點。
- **自行開車**：行經台 8 線（中部橫貫公路）由東側入口進入。
