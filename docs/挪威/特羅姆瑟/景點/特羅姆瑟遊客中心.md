# 特羅姆瑟遊客中心 (Visit Tromsø)

位於特羅姆瑟市中心的遊客中心，在此可以找到報名追極光行程的當地旅行團。

- 地址 Samuel Arnesens gate 5, 9008 Tromsø, Norway
- 電話 +4777610000
- [官方網站](https://www.visittromso.no/)
