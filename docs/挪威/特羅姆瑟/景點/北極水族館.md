# 北極水族館 (Polaria)

北極水族館 (Polaria) 是全球最北端的水族館，位於挪威北部的特羅姆瑟。不同於位於挪威中部的卑爾根的主要科學性質的水族館，Polaria 的設計主要是為了教育體驗，特別強調了針對兒童的展覽。該水族館於 1998 年 5 月開放。

- 地址 Hjalmar Johansens gate 12, 9296 Tromsø, Norway
- 電話 +4777750100
- [官方網站](https://polaria.no/en/)
