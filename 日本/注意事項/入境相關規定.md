# 日本入境規定

## 入境日本前填寫 Visit Japan Web

到日本旅遊時必須使用 Visit Japan Web 填寫並申報資料。
包含入國審查、海關申報及檢疫，使用 QR code 可加速入境時間。

[Visit Japan Web](https://services.digital.go.jp/zh-cmn-hant/visit-japan-web/)

從 2024 年起，入國審查及海關申報都使用同一個 QR code 即可。
